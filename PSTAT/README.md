# PSTAT
茆诗松《概率论与数理统计教程》学习笔记与代码仓库

---

## 仓库说明
本仓库为茆诗松《概率论与数理统计教程》的学习记录，包含各章节笔记、关键公式推导、习题解答与 Python 实现，为后续学习时间序列、机器学习、量化分析打下理论基础。

---

## 目录结构

```text
PSTAT/
├── 01_RANDOM_EVENTS                  # 第1章 随机事件与概率
├── 02_RANDOM_VARIABLES               # 第2章 随机变量及其分布
├── 03_MULTIVARIATE_DISTRIBUTIONS     # 第3章 多维随机变量及其分布
├── 04_LIMIT_THEOREMS                 # 第4章 大数定律与中心极限定理
├── 05_STATISTICS_DISTRIBUTIONS       # 第5章 统计量及其分布
├── 06_PARAMETER_ESTIMATION           # 第6章 参数估计
├── 07_HYPOTHESIS_TESTING             # 第7章 假设检验
├── 08_ANOVA_REGRESSION               # 第8章 方差分析与回归分析
└── README.md                         # 本说明文件
```

---

## 章节学习目标
| 文件夹 | 核心目标 | 关联后续内容 |
| :--- | :--- | :--- |
| 01_RANDOM_EVENTS | 建立概率公理体系，掌握条件概率、贝叶斯公式 | 所有概率模型的基础 |
| 02_RANDOM_VARIABLES | 掌握离散/连续分布，理解随机变量函数分布 | 时序分布、量化模型基础 |
| 03_MULTIVARIATE_DISTRIBUTIONS | 理解联合/边缘/条件分布、协方差与相关系数 | 多变量时序、资产相关性分析 |
| 04_LIMIT_THEOREMS | 掌握大数定律与中心极限定理 | 统计推断、模型收敛性理论 |
| 05_STATISTICS_DISTRIBUTIONS | 掌握样本统计量与三大抽样分布（卡方/t/F） | 假设检验、方差分析的理论支撑 |
| 06_PARAMETER_ESTIMATION | 掌握矩估计、极大似然估计与区间估计 | 时序模型参数估计、GARCH 模型基础 |
| 07_HYPOTHESIS_TESTING | 掌握显著性检验、拟合优度检验、非参数检验 | 时序平稳性检验、模型诊断必备 |
| 08_ANOVA_REGRESSION | 掌握方差分析与一元线性回归 | 时序趋势建模、协整分析前置知识 |

## 工具与依赖
- Python 3.x
- `numpy`：数值计算
- `scipy.stats`：概率分布与统计函数
- `pandas`：数据处理
- `matplotlib`：可视化

---

## 学习说明
1. **笔记格式**：每章文件夹下包含 `notes.md`（知识点与公式）、`exercises.md`（习题解答）、`code.ipynb`（Python 实现）。
2. **代码规范**：所有示例代码可直接运行，附详细注释，便于后续复现。
3. **关联学习**：本仓库与 `TS`（时间序列分析）仓库联动，重点内容会标注对应章节的关联知识点。

---

## 学习顺序建议
1. 优先学习 01-04 章（概率论基础）
2. 再学习 05-07 章（数理统计核心）
3. 08 章可在学习时间序列回归分析时重点回顾
