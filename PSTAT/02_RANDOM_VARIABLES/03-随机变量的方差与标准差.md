# §2.3 随机变量的方差与标准差

## 2.3.1 方差与标准差的定义
- 方差：D(X)=E[(X-E(X))^2]，反映随机变量取值的波动程度。
- 标准差：sigma_X = sqrt(D(X))，与原变量同量纲。
- 简化公式：D(X)=E(X^2)-[E(X)]^2

## 2.3.2 方差的性质
1. D(c)=0（常数无波动）
2. D(aX+b)=a^2D(X)
3. 若 X,Y 独立，则 D(X±Y)=D(X)+D(Y)

### 证明思路

**1. D(c)=0**  
常数 c 的期望 E(c)=c，故 c-E(c)=0，代入定义 D(c)=E[(c-E(c))^2]=E(0)=0。  
也可用简式：D(c)=E(c^2)-[E(c)]^2 = c^2 - c^2 = 0。

**2. D(aX+b)=a^2 D(X)**  
记 mu=E(X)，令 Y=aX+b，则 E(Y)=a*mu+b（期望的线性性），于是

Y - E(Y) = (aX+b) - (a*mu+b) = a(X-mu)。

故 D(Y) = E[(Y-E(Y))^2] = E[a^2 (X-mu)^2] = a^2 E[(X-mu)^2] = a^2 D(X)。  
（常数平移 b 只平移中心，不改变离差平方的波动，故不出现 b。）

**3. 独立时 D(X±Y)=D(X)+D(Y)**  
先证 **D(X+Y)**。记 mu=E(X)，nu=E(Y)。将

X+Y - E(X+Y) = (X-mu) + (Y-nu)

平方后取期望：

D(X+Y) = E[((X-mu)+(Y-nu))^2]
       = E[(X-mu)^2] + E[(Y-nu)^2] + 2 E[(X-mu)(Y-nu)]
       = D(X) + D(Y) + 2 Cov(X,Y)

其中 **Cov(X,Y)=E[(X-mu)(Y-nu)]** 为协方差（与教材记号一致时也可写 Cov(X,Y)=E(XY)-E(X)E(Y)）。  
当 X,Y **独立**时，E(XY)=E(X)E(Y)，故 Cov(X,Y)=0，交叉项消失，得 **D(X+Y)=D(X)+D(Y)**。

再证 **D(X-Y)**。此时

X-Y - E(X-Y) = (X-mu) - (Y-nu)，

平方展开后同样有交叉项 2 E[(X-mu)(-(Y-nu))] = -2 Cov(X,Y)，独立时仍为 0，且 (-(Y-nu))^2=(Y-nu)^2，故

**D(X-Y)=D(X)+D(Y)**（注意：减号下仍是**方差相加**，不是相减）。

**小结**：一般公式为 D(X+Y)=D(X)+D(Y)+2Cov(X,Y)；独立（或更弱地只要 Cov=0，即不相关）时交叉项为 0，得到性质 3。

## 2.3.3 切比雪夫不等式

### 公式（原样）
对任意随机变量 X（只要方差存在），对任意正数 epsilon，有

P(|X - E(X)| >= epsilon) <= D(X) / epsilon^2

（有的书上把 E(X) 记成 mu，把 D(X) 记成 sigma^2，写法不同、意思一样。）

### 用途
只用**期望**和**方差**，就给「偏离均值很远」这类事件的概率做一个**上界**估计；证明极限定理时也会用到。

---

### 符号拆开说（大白话，无推导）

下面把 **epsilon、sigma（方差）、绝对值、P、D(X)** 各自是谁、谁管谁，分开记。

1. **E(X)**（或 mu）：**均值 / 期望**——分布的「中心、平衡点」。

2. **|X - E(X)|**：**随机变量离中心有多远**——绝对值只看距离，不问偏高还是偏低。

3. **epsilon**（读作「伊普西龙」）：**你自己先定的一个「距离门槛」**  
   你想研究：偏离中心**至少多大**算「跑得够远」，这个正数 epsilon 由你取。例如取 5、取 10，都是在定门槛。

4. **|X - E(X)| >= epsilon**：**离中心比门槛还远（含刚好在门槛上）**——「出格、跑太远」这一类结果。

5. **P(...)**：上面这件「出格」的事发生的**概率**。

6. **D(X)**（书上常写成 sigma^2）：**方差**——分布**天生有多散**，由 X 的分布决定，**不是**你随便选的数。

7. **标准差**：sqrt(D(X))，有的书记成 sigma；**方差**是 D(X)=sigma^2。  
   可记：**sigma（标准差）描述波动尺度；epsilon 是你画的一条「警戒线距离」**。

### 整句人话
**随机变量偏离平均值超过你定的门槛 epsilon 的概率，不会超过 D(X) 除以 epsilon 的平方。**

直觉两句：

- 数据**天生越乱**（D(X) 越大），「跑出远门」的概率**上界**越容易大；  
- 你把**门槛拉得越远**（epsilon 越大），「超过门槛」的概率**上界**就越小。

### 口诀
**离均值超门槛的概率 <= 自身方差 / 门槛的平方**

### 极简数字例
设 E(X)=10，D(X)=4；取门槛 epsilon=4。则

P(|X - 10| >= 4) <= 4 / (4^2) = 4/16 = 1/4

人话：**偏离 10 至少为 4 的概率，顶天了也就 25%，不会更高**（注意：这是上界，真实概率可以更小）。

### 易混小结
- **epsilon**：你**人为选定**的距离门槛。  
- **D(X)（sigma^2）**：分布**客观给定**的离散程度（方差）。  
- **不等式在干什么**：用「天生的波动」D(X)，去卡住「越过人为门槛」的概率**最多能有多大**。

## 2.3.4 例子
- 抛一枚均匀硬币，E(X)=0.5，则：
  D(X)=(1-0.5)^2*0.5 + (0-0.5)^2*0.5 = 0.25
