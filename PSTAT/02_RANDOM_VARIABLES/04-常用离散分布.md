# §2.4 常用离散分布



本章把**定义、分布列、期望方差、要点与关系**写成便于对照背诵的形式；记号与茆书习惯一致，公式一律**纯文本**。



---



## 2.4.1 二项分布 B(n,p)



| 项目 | 内容 |

| :--- | :--- |

| **模型** | n 次**独立重复**伯努利试验，每次「成功」概率为 p，「失败」概率为 1-p。 |

| **记号** | X ~ B(n,p) |

| **分布列** | P(X=k) = C(n,k) * p^k * (1-p)^(n-k)，k = 0,1,…,n。这里 C(n,k) 表示从 n 取 k 的组合数。 |

| **期望** | E(X) = n p |

| **方差** | D(X) = n p (1-p) |

| **特例** | n=1 时即为 **0-1 分布（两点分布 / 伯努利分布）**：P(X=1)=p，P(X=0)=1-p。 |

| **例子** | 抛 10 次均匀硬币，正面次数 X ~ B(10, 0.5)。 |

### E(X)=np 与 D(X)=np(1-p) 的证明（标准套路）

**思路**：把 n 次独立试验中的成功次数 X，写成 **n 个相互独立的 0-1 变量之和**。

第 i 次成功令 Yi=1，失败令 Yi=0，则 P(Yi=1)=p，P(Yi=0)=1-p，且 Y1,…,Yn 相互独立，且

X = Y1 + Y2 + … + Yn。

**1. 期望**  
E(Yi) = 1*p + 0*(1-p) = p。由期望的**线性性** E(X)=E(Y1)+…+E(Yn)（此处不必用到独立），得 **E(X)=n p**。

**2. 方差**  
对 0-1 变量，Yi^2=Yi，故 E(Yi^2)=E(Yi)=p，

D(Yi) = E(Yi^2) - [E(Yi)]^2 = p - p^2 = p(1-p)。

各 Yi **相互独立**，独立时方差可加：D(X)=D(Y1)+…+D(Yn)=**n p(1-p)**。

**另注**：也可从分布列出发对 k 求和 k*P(X=k) 并用组合恒等式化简得 np，计算较长；**拆成 n 个独立伯努利之和**最贴合模型，也最常用。

## 2.4.2 泊松分布 P(lambda)



| 项目 | 内容 |

| :--- | :--- |

| **模型** | 固定时间或固定区域内，**稀有事件**发生的次数（电话呼叫、缺陷点数等抽象模型）。 |

| **记号** | X ~ P(lambda)，有的书写作 Poisson(lambda)。 |

| **分布列** | P(X=k) = (lambda^k / k!) * e^(-lambda)，k = 0,1,2,…。 |

| **期望** | E(X) = lambda |

| **方差** | D(X) = lambda（与期望相等，泊松的标志性性质） |

### E(X)=lambda、D(X)=lambda 的证明思路

**期望**：按定义 E(X) = sum_{k=0}^infty k * P(X=k)。k=0 项为 0；k>=1 时 k/k! = 1/(k-1)!，故

E(X) = e^(-lambda) * sum_{k=1}^infty lambda^k/(k-1)!
     = e^(-lambda) * lambda * sum_{m=0}^infty lambda^m/m!   （令 m=k-1）
     = e^(-lambda) * lambda * e^lambda
     = lambda。

**方差**：先算 E[X(X-1)] = sum k(k-1) P(X=k) = lambda^2（同理对级数提取 lambda^2 e^lambda 因子）。又 E(X^2)=E[X(X-1)]+E(X)=lambda^2+lambda，故

D(X) = E(X^2) - [E(X)]^2 = lambda^2 + lambda - lambda^2 = lambda。

### 与二项的关系（泊松近似）

当 n **很大**、p **很小**、且 lambda = n p **适中**时，



B(n,p) 近似于 P(lambda)，其中 lambda = n p。



**常用经验条件（做题参考，非定理唯一条件）**：例如 n >= 100 且 p <= 0.01 一类说法；以教材例题与作业要求为准。

## 2.4.3 超几何分布 H(N,M,n)



| 项目 | 内容 |

| :--- | :--- |

| **模型** | 有限总体共 N 件，其中 M 件为「次品」（或具有某特征），**不放回**抽取 n 件，抽到的次品件数 X。 |

| **记号** | X ~ H(N,M,n)（参数顺序以教材为准：有的书写法为 (n,N,M) 等，读题要核对）。 |

| **分布列** | P(X=k) = C(M,k) * C(N-M, n-k) / C(N,n)。 |

| **取值范围** | k = 0,1,…, min(M,n)，且须满足 n-k <= N-M（非次品够抽）。 |

| **期望** | E(X) = n * (M/N) |

| **方差** | D(X) = n * (M/N) * ((N-M)/N) * ((N-n)/(N-1))。最后因子 (N-n)/(N-1) 为**有限总体修正**。 |

### E(X) 与 D(X) 的证明思路

**期望**：令第 i 次抽到的样品若为次品则 Ii=1，否则 Ii=0，则 X = I1+…+In。每次抽到次品的概率都是 M/N（对称性），故 E(Ii)=M/N；期望线性性给出 **E(X)=n*(M/N)**（**不要求** Ii 独立）。

**方差**：各 Ii 同分布但不独立，不能简单把方差相加。可用**指示变量分解 + 协方差** D(X)=sum D(Ii) + 2*sum_{i<j} Cov(Ii,Ij) 化简得到书上公式；或直接用分布列定义计算。**完整展开以教材为准**，此处只记结论与「有限总体修正因子 (N-n)/(N-1)」的来源。

### 与二项的关系

当总体很大、样本相对很小（常记作 N 远大于 n）时，不放回与放回差别不大，有



H(N,M,n) 近似于 B(n, M/N)，



即把单次抽到次品的概率 p 取为 M/N。

## 2.4.4 几何分布 G(p)（从 1 开始计数）



| 项目 | 内容 |

| :--- | :--- |

| **模型** | 独立重复伯努利试验，每次成功概率 p，**首次成功**出现在第几次试验。 |

| **记号** | X ~ G(p)。注意：有的教材从 **0** 开始计数「失败次数」，分布列形式不同，**以教材定义为准**。 |

| **分布列** | P(X=k) = (1-p)^(k-1) * p，k = 1,2,3,…。 |

| **期望** | E(X) = 1/p |

| **方差** | D(X) = (1-p) / p^2 |

| **无记忆性** | P(X > s+t | X > s) = P(X > t)（在已做 s 次仍未成功的条件下，再等 t 次仍不成功的条件概率，与从头再等 t 次相同）。 |

### E(X)=1/p、D(X)=(1-p)/p^2 的证明思路

记 q=1-p（0<q<1）。**期望**：

E(X) = sum_{k=1}^infty k * q^(k-1) * p = p * sum_{k=1}^infty k q^(k-1)。

而 sum_{k=1}^infty k q^(k-1) 是 sum_{k=0}^infty q^k 对 q 的形式导数，且 sum q^k = 1/(1-q)，求导得 1/(1-q)^2，故 E(X)=p/(1-q)^2 = p/p^2 = **1/p**。

**方差**：可先算 E(X^2)=sum k^2 q^(k-1) p（或对 q 再求导一次），再用 D(X)=E(X^2)-[E(X)]^2 化简得 **(1-p)/p^2**；亦可用负二项 r=1 的结论对照。

## 2.4.5 负二项分布 NB(r,p)（帕斯卡分布）



| 项目 | 内容 |

| :--- | :--- |

| **模型** | 独立重复伯努利试验，每次成功概率 p，**第 r 次成功**出现时，**累计**已做的试验总次数。 |

| **记号** | X ~ NB(r,p)。 |

| **分布列** | P(X=k) = C(k-1, r-1) * p^r * (1-p)^(k-r)，k = r, r+1, r+2,…。 |

| **期望** | E(X) = r / p |

| **方差** | D(X) = r (1-p) / p^2 |

### E(X)=r/p、D(X)=r(1-p)/p^2 的证明思路

设 X 为第 r 次成功时的**总**试验次数。可构造：X = G1 + G2 + … + Gr，其中 G1 为「第 1 次成功」所用次数，G2 为「在第 1 次成功之后再等到第 2 次成功」的**额外**试验次数，…，各段相互独立，且每个 Gi 都服从与 **G(p)**（从 1 计数）相同的几何分布。

于是 **E(X)=r E(G1)=r/p**；独立时 **D(X)=r D(G1)=r(1-p)/p^2**。分布列形式与「负二项= r 个独立几何之和」的模型一致，可与教材定理互证。

### 与几何分布的关系

- r=1 时，NB(1,p) 与 G(p)（本节「从 1 计数」的几何）一致：第 1 次成功即首次成功。  

- **可加性（结论级）**：r 个独立同分布、均服从 G(p) 的「首次成功等待次数」之和，与第 r 次成功时的总试验次数同族（与 NB(r,p) 对应；严格表述以教材定理为准）。



---



## 小结（关系速记）



- **B(1,p)**：0-1 分布（一次试验成败）。  

- **B(n,p)**：n 次独立重复试验中的**成功次数**。  

- **P(lambda)**：稀有事件计数；在 n 大、p 小、lambda=np 适中时，是二项的**极限近似**。  

- **H(N,M,n)**：**有限总体不放回**抽样的次品件数；N 很大时可近似 **B(n, M/N)**。  

- **G(p)**：**首次成功**所需试验次数（本节从 1 计数）；有无记忆性。  

- **NB(r,p)**：**第 r 次成功**时的总试验次数；r=1 退化为几何。

## 期望方差一行对照（背诵用）



| 分布 | E(X) | D(X) |

| :--- | :--- | :--- |

| B(n,p) | n p | n p (1-p) |

| P(lambda) | lambda | lambda |

| H(N,M,n) | n M/N | n (M/N) ((N-M)/N) ((N-n)/(N-1)) |

| G(p) | 1/p | (1-p)/p^2 |

| NB(r,p) | r/p | r(1-p)/p^2 |
