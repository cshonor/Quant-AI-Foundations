# §2.4 常用离散分布

## 2.4.1 二项分布 B(n,p)
- 模型：n次独立重复伯努利试验，每次成功概率为 p。
- 分布列：P(X=k)=C(n,k)*p^k*(1-p)^(n-k)，k=0,1,...,n
- 期望与方差：E(X)=np，D(X)=np(1-p)
- 例子：连续抛10次硬币，正面次数 X ~ B(10,0.5)。

## 2.4.2 泊松分布 P(lambda)
- 模型：描述稀有事件在固定时间/空间内的发生次数。
- 分布列：P(X=k)=(lambda^k/k!)*e^(-lambda)，k=0,1,2,...
- 期望与方差：E(X)=D(X)=lambda

## 2.4.3 超几何分布
- 模型：有限总体不放回抽样，总体 N 件，其中 M 件次品，抽 n 件，次品数 X。
- 分布列：P(X=k)=C(M,k)*C(N-M,n-k)/C(N,n)

## 2.4.4 几何分布与负二项分布
- 几何分布：首次成功所需试验次数。
- 负二项分布：第 r 次成功时的试验次数。
