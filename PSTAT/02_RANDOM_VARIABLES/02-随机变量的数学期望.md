# §2.2 随机变量的数学期望

## 2.2.1 数学期望的概念
- 定义：随机变量的**加权平均值**，反映其取值的中心位置。
- 离散型：E(X)=sum(x_k * p_k)
- 连续型：E(X)=从 -∞ 到 +∞ 的 x*f(x) 对 x 的积分

## 2.2.2 数学期望的性质
1. E(c)=c（常数的期望是自身）
2. E(aX+b)=aE(X)+b（线性性质）
3. E(X+Y)=E(X)+E(Y)（和的期望=期望的和）
4. 若 X,Y 独立，则 E(XY)=E(X)E(Y)

## 2.2.3 例子
- 抛一枚均匀硬币，X=1（正面），X=0（反面）：
  E(X)=1*0.5 + 0*0.5 = 0.5
