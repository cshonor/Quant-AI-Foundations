# §2.2 随机变量的数学期望

## 2.2.1 数学期望的概念
- 定义：随机变量的**加权平均值**，反映其取值的中心位置。
- 离散型：E(X)=sum(x_k * p_k)
- 连续型：E(X)=从 -∞ 到 +∞ 的 x*f(x) 对 x 的积分

### 按定义计算的例子

**离散例 1（均匀骰子）** 设 P(X=k)=1/6，k=1,2,…,6，则

E(X) = 1*(1/6) + 2*(1/6) + … + 6*(1/6) = (1+2+…+6)/6 = 21/6 = 3.5。

**离散例 2（均匀硬币）** P(X=1)=P(X=0)=0.5 时，E(X)=1*0.5+0*0.5=0.5（与 2.2.3 中硬币例题同一题）。

**连续例 1（[0,1] 上均匀）** f(x)=1（当 0<x<1），其余为 0，则

E(X) = 从 0 到 1 对 x*1 积分 = [x^2/2]_0^1 = 1/2。

**连续例 2（指数分布，参数 lambda>0）** f(x)=lambda*e^(-lambda x)（x>0），可算得 E(X)=1/lambda（分部积分或查表；细节见「常用连续分布」）。

## 2.2.2 数学期望的性质
1. E(c)=c（常数的期望是自身）
2. E(aX+b)=aE(X)+b（线性性质）
3. E(X+Y)=E(X)+E(Y)（和的期望=期望的和）
4. 若 X,Y 独立，则 E(XY)=E(X)E(Y)

### 证明思路（与教材定义对齐；离散写求和，连续写积分）

**1. E(c)=c**  
把常数 c 看成随机变量：以概率 1 取 c，即 P(X=c)=1。离散定义下 E(X)=c*1=c。

**2. E(aX+b)=aE(X)+b**  
离散：E(aX+b)=sum_k (a x_k + b) p_k = a*sum_k x_k p_k + b*sum_k p_k = a E(X) + b（用到 sum_k p_k=1）。  
连续：E(aX+b)=从 -∞ 到 +∞ (a x + b) f(x) dx = a*从 -∞ 到 +∞ x f(x) dx + b*从 -∞ 到 +∞ f(x) dx = a E(X) + b。

**3. E(X+Y)=E(X)+E(Y)**（**不要求** X,Y 独立）  
离散（联合分布列 p_ij = P(X=x_i, Y=y_j)）：

E(X+Y) = sum_i sum_j (x_i + y_j) p_ij
       = sum_i sum_j x_i p_ij + sum_i sum_j y_j p_ij
       = sum_i x_i (sum_j p_ij) + sum_j y_j (sum_i p_ij)
       = sum_i x_i P(X=x_i) + sum_j y_j P(Y=y_j)
       = E(X) + E(Y)。

连续：E(X+Y)=二重积分 (x+y) f(x,y) dx dy，拆成两个积分分别得到 E(X) 与 E(Y)。

**4. 独立时 E(XY)=E(X)E(Y)**  
离散：独立时 p_ij = P(X=x_i)P(Y=y_j)，故

E(XY) = sum_i sum_j (x_i y_j) p_ij
      = sum_i sum_j x_i y_j P(X=x_i) P(Y=y_j)
      = (sum_i x_i P(X=x_i)) (sum_j y_j P(Y=y_j))
      = E(X) E(Y)。

连续：独立时联合密度 f(x,y)=f_X(x) f_Y(y)，二重积分 x y f(x,y) dx dy 可分离为两个一维积分之积，即 E(X)E(Y)。

**注意**：性质 4 的证明用到了「联合分布 = 边缘分布的乘积」。没有独立时，E(XY) 须按联合分布用二重求和或二重积分计算，**一般不能**写成 E(X)E(Y)（独立只是使该式成立的常用充分条件之一）。

## 2.2.3 例子
- 抛一枚均匀硬币，X=1（正面），X=0（反面）：
  E(X)=1*0.5 + 0*0.5 = 0.5
