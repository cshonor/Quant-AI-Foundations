# §2.5 常用连续分布

本章给出**记号、密度、期望方差、核心性质与分布间关系**，公式一律**纯文本**，便于背诵与对照。参数记号以常见教材为准；若与某书「形状—尺度」写法不同，读题时先核对定义。

---

## 2.5.1 正态分布 N(mu, sigma^2)

| 项目 | 内容 |
| :--- | :--- |
| **记号** | X ~ N(mu, sigma^2)，其中 mu 为实数，sigma > 0（有的书把 sigma^2 记作方差、sigma 记作标准差）。 |
| **密度** | f(x) = [1 / (sigma * sqrt(2*pi))] * e^(-(x-mu)^2 / (2*sigma^2))，x 取全体实数。 |
| **期望** | E(X) = mu |
| **方差** | D(X) = sigma^2 |
| **标准正态** | N(0,1)：密度常记为 phi(x)，分布函数记为 Phi(x)。一般正态可通过标准化 Z=(X-mu)/sigma ~ N(0,1) 查表或数值计算。 |
| **3σ 经验法则** | 约 **68%** 质量落在 (mu-sigma, mu+sigma)；约 **95%** 落在 mu 加减 2sigma 内；约 **99.7%** 落在 mu 加减 3sigma 内（精确数值随定义区间开闭略有差别，以教材表为准）。 |
| **用途** | 测量误差、大量独立小因素叠加的近似模型；**中心极限定理**下样本均值等极限分布。 |

---

## 2.5.2 均匀分布 U(a,b)

| 项目 | 内容 |
| :--- | :--- |
| **记号** | X ~ U(a,b)，要求 a < b。 |
| **密度** | 当 a < x < b 时，f(x) = 1/(b-a)；其余 x 处 f(x)=0。 |
| **期望** | E(X) = (a + b) / 2 |
| **方差** | D(X) = (b-a)^2 / 12 |
| **标准均匀** | U(0,1) 常用于随机模拟、蒙特卡洛与「概率积分变换」的起点。 |

---

## 2.5.3 指数分布 Exp(lambda)

| 项目 | 内容 |
| :--- | :--- |
| **记号** | X ~ Exp(lambda)，速率参数 lambda > 0（有的书用尺度 1/lambda，**读题核对**）。 |
| **密度** | f(x) = lambda * e^(-lambda * x)，当 x > 0；x <= 0 时 f(x)=0。 |
| **期望** | E(X) = 1 / lambda |
| **方差** | D(X) = 1 / lambda^2 |
| **无记忆性** | P(X > s+t | X > s) = P(X > t)（s,t>0），与离散几何分布的无记忆性平行。 |
| **用途** | 寿命、等待时间；**泊松过程**中事件间隔时间（强度 lambda 时）常建模为 Exp(lambda)。 |

---

## 2.5.4 伽马分布 Gamma(alpha, lambda)

| 项目 | 内容 |
| :--- | :--- |
| **记号** | X ~ Gamma(alpha, lambda)，形状参数 alpha > 0，速率参数 lambda > 0（与指数分布同一套速率约定时）。 |
| **密度** | 当 x > 0 时，f(x) = [lambda^alpha / Gamma(alpha)] * x^(alpha-1) * e^(-lambda * x)；x <= 0 时 f(x)=0。 |
| **Gamma 函数** | Gamma(alpha) = 从 0 到 +∞ 对 t^(alpha-1) * e^(-t) dt；递推 **Gamma(alpha+1) = alpha * Gamma(alpha)**；Gamma(1)=1，Gamma(n)=(n-1)!（n 为正整数）。 |
| **期望** | E(X) = alpha / lambda |
| **方差** | D(X) = alpha / lambda^2 |
| **特例** | alpha=1 时 **Gamma(1, lambda) = Exp(lambda)**。 |
| **与卡方** | **卡方 chi-square(n)** 可视为 **Gamma(n/2, 1/2)**（与常见密度写法一致时；不同参数化会差常数因子，**以教材定义为准**）。 |
| **用途** | 多个独立 Exp(lambda) 等待时间之和；排队、可靠性；为卡方、t 等抽样分布提供分析基础。 |

---

## 2.5.5 贝塔分布 Beta(alpha, beta)

（**Beta** 为分布名称；**alpha、beta** 为两个形状参数，与希腊字母同名，读式子时分清「分布 Beta」与「参数 beta」。）

| 项目 | 内容 |
| :--- | :--- |
| **记号** | X ~ Beta(alpha, beta)，alpha > 0，beta > 0。 |
| **密度** | 当 0 < x < 1 时，f(x) = [Gamma(alpha+beta) / (Gamma(alpha)*Gamma(beta))] * x^(alpha-1) * (1-x)^(beta-1)；区间外为 0。 |
| **期望** | E(X) = alpha / (alpha + beta) |
| **方差** | D(X) = alpha * beta / [ (alpha+beta)^2 * (alpha+beta+1) ] |
| **特例** | alpha=1 且 beta=1 时 **Beta(1,1) = U(0,1)**（(0,1) 上均匀）。 |
| **用途** | (0,1) 上比例、概率的随机建模；二项成功概率的**共轭先验**（贝叶斯统计，见第 6 章笔记）。 |

---

## 小结（关系速记）

- **Exp(lambda) = Gamma(1, lambda)**。  
- **chi-square(n)** 与 **Gamma(n/2, 1/2)** 同一族（参数化与教材对齐）。  
- **U(0,1) = Beta(1, 1)**。  
- **正态**：对称、钟形，CLT 极限。  
- **均匀**：有限区间内「等概率密度」。  
- **指数**：正半轴、无记忆、泊松间隔。  
- **伽马**：指数与卡方的推广，独立指数和。  
- **贝塔**：有界区间 (0,1)、比例与贝叶斯先验。

## 期望方差一行对照（背诵用）

| 分布 | E(X) | D(X) |
| :--- | :--- | :--- |
| N(mu, sigma^2) | mu | sigma^2 |
| U(a,b) | (a+b)/2 | (b-a)^2/12 |
| Exp(lambda) | 1/lambda | 1/lambda^2 |
| Gamma(alpha, lambda) | alpha/lambda | alpha/lambda^2 |
| Beta(alpha, beta) | alpha/(alpha+beta) | alpha*beta/[(alpha+beta)^2(alpha+beta+1)] |
