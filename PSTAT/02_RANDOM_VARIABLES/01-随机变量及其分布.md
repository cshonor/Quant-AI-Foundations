# §2.1 随机变量及其分布

## 2.1.1 随机变量的概念
- 定义：定义在样本空间 Omega 上的实值函数 X = X(w)，其中 w 属于 Omega。
- 本质：把随机试验的结果“数值化”，让事件变成可计算的数。
- 例子：抛硬币，定义 X=1（正面），X=0（反面），则 X 就是一个随机变量。

## 2.1.2 随机变量的分布函数
- 定义：F(x) = P(X<=x)，其中 x 属于实数集 R。
- 性质：
  1. 单调不减
  2. 右连续
  3. F(-∞)=0，F(+∞)=1
- 用途：描述随机变量的整体概率分布情况。

## 2.1.3 离散型随机变量的概率分布列
- 定义：P(X=x_k)=p_k，k=1,2,...，满足：
  - p_k >= 0
  - sum(p_k)=1
- 例子：抛一枚均匀硬币，分布列为：

| X | 0 | 1 |
|---|---|---|
| P | 0.5 | 0.5 |

## 2.1.4 连续型随机变量的概率密度函数
- 定义：存在非负函数 f(x)，使得 F(x)=从 -∞ 到 x 的 f(t) 对 t 的积分，则 X 为连续型随机变量，f(x) 为其概率密度函数。
- 性质：
  1. f(x) >= 0
  2. 从 -∞ 到 +∞ 的 f(x) 积分等于 1
- 注意：连续型随机变量在单点的概率为 P(X=a)=0。
