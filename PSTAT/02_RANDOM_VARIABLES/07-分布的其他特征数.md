# §2.7 分布的其他特征数

## 2.7.1 k阶矩
- 原点矩：E(X^k)
- 中心矩：E[(X-E(X))^k]
- 期望是一阶原点矩，方差是二阶中心矩。

## 2.7.2 变异系数
- 定义：Cv = sqrt(D(X)) / |E(X)|，消除量纲影响，比较不同量纲数据的波动。

## 2.7.3 分位数
- 定义：p 分位数 x_p 满足 F(x_p)=p。
- 用途：构造置信区间、异常值检测。

## 2.7.4 中位数
- 0.5分位数，抗干扰性比均值强。

## 2.7.5 偏度系数
- 定义：Skew = E[(X-E(X))^3] / D(X)^(3/2)，描述分布的对称性。

## 2.7.6 峰度系数
- 定义：Kurt = E[(X-E(X))^4] / D(X)^2，描述分布尾部的厚薄。
