# 5.3 统计量及其分布

## 本节定位
**统计量**是样本的不含未知参数的函数，是估计与检验的载体；其分布称为**抽样分布**（与总体分布不同）。

---

## 一、定义
设 T = T(X_1, X_2, ..., X_n)。若 T 的表达式中**不含有任何未知参数**，则称 T 为一个**统计量**。

### 例子（正态总体）
设 X ~ N(mu, sigma^2)，且 mu 未知。

- **不是统计量**：X_bar - mu（仍含未知 mu）  
- **是统计量**：样本均值 X_bar = (1/n) * 对 i=1 到 n 求和 X_i（只由样本构成）

---

## 二、常用统计量（务必分清分母）
以下均假定样本为 X_1,...,X_n，观测时用小写亦可。

### 1. 样本均值
X_bar = (1/n) * 对 i=1 到 n 求和 X_i

### 2. 样本方差（无偏方差，分母 n-1）
S^2 = (1/(n-1)) * 对 i=1 到 n 求和 (X_i - X_bar)^2

### 3. 样本标准差
S = sqrt(S^2)

### 4. 样本 k 阶原点矩
A_k = (1/n) * 对 i=1 到 n 求和 X_i^k

### 5. 样本 k 阶中心矩
B_k = (1/n) * 对 i=1 到 n 求和 (X_i - X_bar)^k

说明：S^2 用 n-1 是为了在正态等模型下对 sigma^2 **无偏**；与「除以 n 的样本二阶中心矩」不是同一个量。

---

## 三、「统计量的分布」指什么？
给定总体分布后，T(X_1,...,X_n) 作为随机变量的分布，称为 T 的**抽样分布**。  
例如正态总体下 X_bar 的精确分布、S^2 与卡方的关系等，见下一节「三大抽样分布」。

---

## 关键小结
- 统计量 = 样本的函数 + **不含未知参数**  
- X_bar、S^2、S、A_k、B_k 是最常用的「原材料」  
- 抽样分布是第 6、7 章区间估计与假设检验的底座
