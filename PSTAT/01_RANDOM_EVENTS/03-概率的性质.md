# §1.3 概率的性质

## 一、基础简单性质
由概率三公理直接推出：

1. 不可能事件概率
P(empty set) = 0

2. 有界性
对任意随机事件 A：
0 <= P(A) <= 1

## 二、对立事件概率公式
P(非A) = 1 - P(A)

### 经典例子
掷骰子，求“点数不是6”的概率。
- 设 A：点数为6，P(A)=1/6

P(非A) = 1 - 1/6 = 5/6

> 遇到“不发生、至少一个不、反面”，优先用对立公式简化计算。

## 三、单调性
若 A 是 B 的子集（A发生必导致B发生），则：
1. 概率大小：P(A) <= P(B)
2. 差事件公式：P(B-A) = P(B)-P(A)

### 例子
A：点数为2，B：点数为偶数
显然 A 是 B 的子集，偶数概率大于单点概率，符合单调性。

## 四、有限可加性
若 A1, A2, ..., An **两两互斥**，则：
P(A1并A2并...并An) = P(A1)+P(A2)+...+P(An)

白话：互斥事件，“至少一个发生”的概率 = 各自概率直接相加。

## 五、概率加法公式（重点必考）
### 1. 两个事件加法公式
P(A并B) = P(A) + P(B) - P(AB)

- 为什么要减 P(AB)？
  交集部分被重复加了一次，需要扣掉重复部分。

### 生活化例子
全班同学，A=喜欢数学，B=喜欢英语
喜欢数学 **或** 英语的人数 = 喜欢数学 + 喜欢英语 - 两门都喜欢

### 2. 三个事件加法公式
P(A并B并C)
= P(A)+P(B)+P(C)- P(AB)-P(AC)-P(BC)+ P(ABC)
容斥原理思想：先加、再减重复、再加回多减的部分。

## 六、概率的连续性（了解+理论铺垫）
针对单调递增/递减事件列：
- 递增事件列：概率极限 = 极限事件的概率
- 递减事件列：概率极限 = 极限事件的概率

作用：连接**概率 + 极限**，是后面大数定律、中心极限定理、测度论概率的理论基础。
