# 第 3 章笔记：线性回归

## 本节核心

- 线性回归是一种常用的数值预测方法，用来估计连续值。  
- 典型应用：房价、股票价值、寿命、用户观看时长等。  
- 目标是找到一条“尽量贴近所有数据点”的直线。  

## 直观理解

- 图中散点代表观测数据，整体呈现近似线性趋势。  
- 我们希望画出一条线，让它与这些点的整体距离最小。  
- 这条线就是线性回归模型对数据规律的近似表达。  

## 类比（书中思路）

- 把散点看成城市中的房屋位置。  
- 画出的回归线像一条穿过城市的道路。  
- 道路越靠近大多数房屋，说明这条线越“符合需求”。  

## 一句话总结

> 线性回归就是用一条直线去近似数据中的变化趋势，并据此做连续值预测。

## 3.1 问题：预测房屋价格

### 场景

- 假设我们是房地产经纪人，需要给新房定价。  
- 房价会受很多特征影响：面积、房间数、位置、犯罪率、学校质量、到市中心距离等。  
- 目标是找到一个函数，用这些特征去预测新房价格。  

## 3.2 解决方案：建立房价回归模型

### 简化建模示例（单特征）

- 为了先理解核心思路，先只用一个特征：**房间数**。  
- 已知样本（房间数 -> 价格，单位万美元）：  
  - 1 -> 15  
  - 2 -> 20  
  - 3 -> 25  
  - 4 -> ?（待预测）  
  - 5 -> 35  
  - 6 -> 40  
  - 7 -> 45  

### 从样本中观察规律

- 每增加 1 个房间，价格大约增加 5 万美元。  
- 可理解为：房子有一个“基础价格”，再叠加“每个房间带来的增量价格”。  
- 这就是线性回归的核心：用线性关系描述特征与目标值之间的变化。  

### 一句话总结

> 在线性回归里，我们先从数据中找到“每增加一点特征，目标值大致变化多少”的规律，再用这个规律做预测。
