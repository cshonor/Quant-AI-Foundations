## 本章核心目标
理解频谱分析的基本原理，学会用傅里叶变换和滤波方法处理时间序列。

## 关键概念
1. **循环行为与周期性**：序列中的重复波动模式
2. **谱密度**：频域下的功率分布，揭示序列的主导周期
3. **傅里叶变换**：将时域序列分解为不同频率的正弦波叠加
4. **线性滤波器**：低通/高通/带通滤波，提取特定频率的信号
5. **信号提取与最佳滤波**：维纳滤波基础，最大化信噪比

## 量化应用场景
- 识别股市的周度、月度、季度周期效应
- 对因子序列做低通滤波，去除高频噪声，保留中长期趋势
- 挖掘周期类量价因子，构建周期轮动策略

## 实战练习
```python
# 1. 用傅里叶变换分析序列周期
import numpy as np
import matplotlib.pyplot as plt

# 假设ret是收益率序列
n = len(ret)
freq = np.fft.fftfreq(n, d=1)
fft_vals = np.fft.fft(ret)
power = np.abs(fft_vals) ** 2

plt.plot(freq, power)
plt.title('功率谱密度')
plt.show()
```
