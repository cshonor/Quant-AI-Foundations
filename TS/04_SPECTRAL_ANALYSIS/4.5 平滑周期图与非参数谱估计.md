# 4.5 平滑周期图与非参数谱估计

## 核心思路
在原始周期图的基础上，对**相邻频率点做局部平均**或分段平均，压低毛刺，得到更光滑的谱估计。

这类方法多属**非参数**：不强设序列必为某阶 AR/MA，主要由数据驱动。

---

## 操作逻辑（概念）
1. 计算原始周期图或等价的 FFT 功率分配  
2. 在频率邻域做滑动平均，或分段（如 Welch）再平均  
3. 得到更平滑的谱曲线，便于读峰与带宽

---

## 优缺点
- **优点**：不绑死参数模型，通用、直观  
- **缺点**：平滑越强，**频率分辨率越差**，可能抹掉窄而真的峰

---

## Python：Welch 法（平滑谱估计）
`scipy.signal` 中的函数名为小写 **`welch`**。

```python
import matplotlib.pyplot as plt
from scipy.signal import welch

f, Pxx_smooth = welch(x, fs=1, nperseg=min(256, len(x) // 4))
plt.plot(f, Pxx_smooth)
plt.title("Welch 平滑谱估计")
plt.xlabel("频率 f")
plt.ylabel("功率谱密度（估计）")
plt.show()
```
