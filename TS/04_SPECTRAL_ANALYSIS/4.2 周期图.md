# 4.2 周期图（Periodogram）

## 定义
对零均值（或已去均值）的平稳序列样本 x_1,...,x_n，周期图在角频率 omega 处可写为：

I(omega) = (1/n) * | 对 t=1 到 n 求和 [ x_t * exp(-i * omega * t) ] |^2

其中 i 为虚数单位；实务中由 FFT 计算离散频率上的值。

---

## 人话解释
对序列做离散傅里叶变换，再取模平方并做尺度归一，得到**各频率上的能量强度**。  
哪个频率对应的值高，说明样本里该频率附近的波动更突出。

---

## 关键性质（记要点）
1. 常用角频率范围：omega 属于 [0, pi]（与采样设定对应）  
2. 低频多对应慢变化、长周期；高频多对应短周期与噪声  
3. **原始周期图曲线很毛**、方差大，一般**不直接**当作最终谱密度估计使用，需要平滑或参数模型

---

## Python 实操（scipy）
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import periodogram

# x 为一维数组，fs 为采样率；日收益等若按「每期=1单位时间」可设 fs=1
f, Pxx = periodogram(x, fs=1)
plt.plot(f, Pxx)
plt.title("原始周期图")
plt.xlabel("频率 f")
plt.ylabel("功率")
plt.show()
```
