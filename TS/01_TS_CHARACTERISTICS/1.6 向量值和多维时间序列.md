# 1.6 多维时间序列基础与交叉相关分析

## 一、核心定义
多维时间序列是单变量时间序列的扩展，记为：
x_t = (x_1t, x_2t, ..., x_kt)^T

- 每个分量 x_it 本身都是一条单变量时间序列
- 例：同时观测价格、成交量、波动率，可构成三维时间序列

下面用「数据表」类比，把同一套对象一眼对齐到表格里的行与列。

---

## 二、多维时间序列，本质就是一张「数据表」
可以直接看成一张**二维数据表**：

| 时间 t | 变量1（价格）x_1t | 变量2（成交量）x_2t | 变量3（波动率）x_3t |
|--------|-------------------|---------------------|---------------------|
| 1      | 100               | 5000                | 0.02                |
| 2      | 101               | 6000                | 0.03                |
| 3      | 99                | 4500                | 0.01                |
| ...    | ...               | ...                 | ...                 |

- **行**：不同时间点 t；每一行就是一个向量 x_t = (x_1t, x_2t, x_3t)^T，与上文定义一致。
- **列**：不同变量（维度）；每一列单独抽出来，就是一条单变量时间序列（价格序列、成交量序列等），与「每个分量 x_it 是一条单变量序列」一一对应。

---

## 三、笔记里的概念，在表里是什么？

### 1) 均值向量 mu = E(x_t)
对**每一列**求时间平均（样本里用整段样本均值近似期望），再按列顺序排成向量：
mu = (mu_1, mu_2, mu_3)^T

- mu_1：价格列的均值
- mu_2：成交量列的均值
- 以此类推

### 2) k 阶自协方差矩阵 Gamma(k) = E[(x_t - mu)(x_{t+k} - mu)^T]
矩阵里每个位置，都是「某两列」在滞后 k 下的协方差（同一列就是自协方差）：

- **对角线元素**：同一列与自身滞后 k 的协方差，即该单变量序列的 lag-k 自协方差。
- **非对角线元素**：两列之间的**交叉协方差**，例如价格与成交量在间隔 k 上的联动强度。

### 3) 交叉相关函数（CCF）
专门刻画**两列**之间的相关：在样本里对 (x_t, y_{t-k}) 或 (x_t, y_{t+k}) 做估计并标准化，得到 rho_xy(k)，用来判断「A 是否领先/滞后于 B」「同向还是反向」等（与 1.3、1.5 节单变量 CCF 定义衔接）。

---

## 四、为什么叫「多维时间序列」而不只说「一张表」？
强调两点，和普通「只有行号没有先后逻辑」的表不同：

1. **时间是主轴**：行必须按时间顺序读，不是 i.i.d. 横截面样本；建模、预测、检验都依赖这条时间轴。
2. **列与列之间要联动分析**：不只单独看每一列，还要看列与列的动态关系（成交量、波动率是否领先价格等），这正是多维时序区别于「各列互不相干地当横截面」的地方。

---

## 五、多维模型基础：向量自回归（VAR）
一阶向量自回归 VAR(1) 可写为：
x_t = Phi * x_{t-1} + w_t

- 其中 Phi 是系数矩阵，w_t 是向量噪声项
- 矩阵形式可刻画多个序列之间的动态联动与相互影响

---

## 六、量化交易中的意义
1. 配对交易、多因子模型，本质都在处理多维时间序列关系。
2. CCF 可用于识别领先指标（如成交量、波动率对价格的领先效应）。
3. 是多变量时序建模、宏观因子联动分析、VAR 实务应用的基础。

---

## 七、Python 小例子：建表、均值向量、样本协方差
下面用三列模拟「价格、成交量、波动率」，便于你本地跑一遍，把「表」与「向量」对上号。

```python
import numpy as np
import pandas as pd

np.random.seed(0)
n = 5
# 模拟三列时序（仅演示结构，不是真实行情生成过程）
price = 100 + np.cumsum(np.random.randn(n))
volume = 5000 + 200 * np.random.randn(n)
vol = 0.02 + 0.005 * np.random.rand(n)

df = pd.DataFrame(
    {"t": np.arange(1, n + 1), "x_1_price": price, "x_2_volume": volume, "x_3_vol": vol}
)
X = df[["x_1_price", "x_2_volume", "x_3_vol"]]

# 均值向量：按列平均
mu_hat = X.mean()
print("数据表：")
print(df)
print("\n均值向量 mu_hat（每列样本均值）：")
print(mu_hat)

# 样本协方差矩阵（lag 0）：pandas 默认是列与列之间的协方差
Gamma0_hat = X.cov()
print("\n样本协方差矩阵 Gamma_hat(0)（列协方差）：")
print(Gamma0_hat)

# 两列之间的样本 CCF 可用 statsmodels 的 ccf；此处仅提示与 1.3 节公式一致
# from statsmodels.tsa.stattools import ccf
```

说明：
- 上面对 Gamma(0) 用的是**列与列同时刻**的样本协方差矩阵；滞后 k 的 Gamma(k) 需要对 (x_t - mu) 与 (x_{t+k} - mu) 做跨期配对，实现上可用自定义循环或专门的多变量时序包，思路与 1.3 节标量 gamma(k) 相同，只是把标量换成矩阵。
