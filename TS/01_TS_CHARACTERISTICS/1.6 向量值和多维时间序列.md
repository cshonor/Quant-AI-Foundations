# 1.6 多维时间序列基础与交叉相关分析

## 一、核心定义
多维时间序列是单变量时间序列的扩展，记为：
x_t = (x_1t, x_2t, ..., x_kt)^T

- 每个分量 x_it 本身都是一条单变量时间序列
- 例：同时观测价格、成交量、波动率，可构成三维时间序列

下面用「数据表」类比，把同一套对象一眼对齐到表格里的行与列。

---

## 二、多维时间序列，本质就是一张「数据表」
可以直接看成一张**二维数据表**：

| 时间 t | 变量1（价格）x_1t | 变量2（成交量）x_2t | 变量3（波动率）x_3t |
|--------|-------------------|---------------------|---------------------|
| 1      | 100               | 5000                | 0.02                |
| 2      | 101               | 6000                | 0.03                |
| 3      | 99                | 4500                | 0.01                |
| ...    | ...               | ...                 | ...                 |

- **行**：不同时间点 t；每一行就是一个向量 x_t = (x_1t, x_2t, x_3t)^T，与上文定义一致。
- **列**：不同变量（维度）；每一列单独抽出来，就是一条单变量时间序列（价格序列、成交量序列等），与「每个分量 x_it 是一条单变量序列」一一对应。

---

## 三、笔记里的概念，在表里是什么？

### 1) 均值向量 mu = E(x_t)
对**每一列**求时间平均（样本里用整段样本均值近似期望），再按列顺序排成向量：
mu = (mu_1, mu_2, mu_3)^T

- mu_1：价格列的均值
- mu_2：成交量列的均值
- 以此类推

**小例子（均值向量）**  
只看两列、三个时间点（单位随意，只为算数）：

| t | x_1t（价格） | x_2t（量） |
|---|-------------|-----------|
| 1 | 10          | 100       |
| 2 | 12          | 120       |
| 3 | 11          | 110       |

- mu_1 = (10 + 12 + 11) / 3 = 11  
- mu_2 = (100 + 120 + 110) / 3 = 110  
- 均值向量写成 mu = (11, 110)^T（与「对每一列求时间平均」完全一致）

### 2) k 阶自协方差矩阵 Gamma(k) = E[(x_t - mu)(x_{t+k} - mu)^T]
矩阵里每个位置，都是「某两列」在滞后 k 下的协方差（同一列就是自协方差）：

- **对角线元素**：同一列与自身滞后 k 的协方差，即该单变量序列的 lag-k 自协方差。
- **非对角线元素**：两列之间的**交叉协方差**，例如价格与成交量在间隔 k 上的联动强度。

**和 CCF 是不是一回事？**  
不完全是。**Gamma(k) 里每个位置先是协方差** gamma_ij(k)（i 不等于 j 为交叉协方差，i 等于 j 为自协方差），**还不是 CCF 本身**。  
**CCF（互相关）一般指相关系数** rho_ij(k)：在常见定义下，用 gamma_ij(k) 除以 sqrt(gamma_ii(0) * gamma_jj(0))（或与你所用教材完全等价的另一种标准化），得到无量纲、大致落在 [-1, 1] 的量，才好比较量纲不同的两列。  
一句话：**同一滞后 k、同一对列 (i,j)，先写在矩阵里的是协方差；CCF 是它的标准化版本**，和 ACF 与自协方差的关系是同一类区别。

**小例子（lag 0 的 2 乘 2 矩阵，样本协方差）**  
仍用上面两列三行，先把每列减去自己的均值：

- x_1 去均值：-1, 1, 0  
- x_2 去均值：-10, 10, 0  

样本协方差用分母 (n-1)=2（与 pandas `ddof=1` 一致）：

- 对角 (1,1)：((-1)^2 + 1^2 + 0^2) / 2 = 1  
- 对角 (2,2)：((-10)^2 + 10^2 + 0^2) / 2 = 100  
- 非对角 (1,2) 与 (2,1)：((-1)(-10) + (1)(10) + 0) / 2 = 10  

于是 lag 0 的样本矩阵可记为：

- Gamma_hat(0) = 第一行 (1, 10)，第二行 (10, 100)  
  即：同一时刻「价格与成交量」同涨同跌时，非对角元为正，表示正向联动。

**顺带一句 lag k 怎么看表**  
若 k=1，矩阵里 (1,2) 位置对应的是「t 时刻第 1 列」与「t+1 时刻第 2 列」配对后求协方差（表里要错一行对齐），不再是对角线上「同一列自己和自己」那么简单；对角线上 (i,i) 仍是第 i 列与自身滞后 k 的自协方差。

#### 补充：五乘五 Gamma(k) 通俗实例（五个金融列）
先约定向量里五个分量（每一列都是一条时序，t 为交易日）：

1. x_1：股票收盘价  
2. x_2：成交量  
3. x_3：涨跌幅  
4. x_4：换手率  
5. x_5：北向资金净流入  

滞后阶数为 k，矩阵定义（与上文一致，写成纯文本）：

Gamma(k) = E[ (x_t - mu) * (x_{t+k} - mu)^T ]

其中 x_t = (x_1t,...,x_5t)^T，mu 为五维均值向量；乘法是列向量乘行向量，得到五乘五矩阵。

**五乘五矩阵长什么样（元素记号）**  
第 i 行、第 j 列元素记为 gamma_ij(k)，整块矩阵可理解为：

```
Gamma(k) =
[ gamma_11(k)  gamma_12(k)  gamma_13(k)  gamma_14(k)  gamma_15(k) ]
[ gamma_21(k)  gamma_22(k)  gamma_23(k)  gamma_24(k)  gamma_25(k) ]
[ gamma_31(k)  gamma_32(k)  gamma_33(k)  gamma_34(k)  gamma_35(k) ]
[ gamma_41(k)  gamma_42(k)  gamma_43(k)  gamma_44(k)  gamma_45(k) ]
[ gamma_51(k)  gamma_52(k)  gamma_53(k)  gamma_54(k)  gamma_55(k) ]
```

**一、对角线（自己对自己，自协方差）**  
- gamma_11(k)：t 时刻价格 与 t+k 时刻价格的协方差  
- gamma_22(k)：t 时刻成交量 与 t+k 时刻成交量的协方差  
- gamma_33(k)：t 时刻涨跌幅 与 t+k 时刻涨跌幅的协方差  
- gamma_44(k)：t 时刻换手率 与 t+k 时刻换手率的协方差  
- gamma_55(k)：t 时刻北向净流入 与 t+k 时刻北向净流入的协方差  

对角线就是单变量里 lag-k 自协方差；若再除以该列的 lag-0 方差开根号，就接到单变量 ACF 那一套（1.3、1.5 节）。

**二、非对角线（A 对 B，跨列跨期，交叉协方差）**  
举几个位置帮助你对号入座（读法都是「左下标列在 t，右下标列在 t+k」这一套约定，与教材写法一致即可）：

- gamma_12(k)：t 时刻价格 与 t+k 时刻成交量的协方差  
- gamma_13(k)：t 时刻价格 与 t+k 时刻涨跌幅的协方差  
- gamma_25(k)：t 时刻成交量 与 t+k 时刻北向净流入的协方差  
- gamma_51(k)：t 时刻北向净流入 与 t+k 时刻价格的协方差  

非对角线刻画多变量之间跨期联动；对两列做标准化后，就接到 CCF、互相关那一层（与下面第 3 点衔接）。

**三、几条串联知识**  
1. Gamma(0)：k=0，即「同一时刻」五个变量之间的协方差矩阵（同期相关结构）。  
2. 宽平稳下常见对称关系：gamma_ij(k) = gamma_ji(-k)（记号上注意 k 与 -k 对应「谁领先谁」）。  
3. 把每个 gamma_ij(k) 用各自尺度归一化（除以 sqrt(gamma_ii(0)*gamma_jj(0)) 一类），得到相关系数矩阵，对应多变量视角下的自相关、互相关（ACF、CCF）。

### 3) 交叉相关函数（CCF）
专门刻画**两列**之间的相关：在样本里对 (x_t, y_{t-k}) 或 (x_t, y_{t+k}) 做估计并标准化，得到 rho_xy(k)，用来判断「A 是否领先/滞后于 B」「同向还是反向」等（与 1.3、1.5 节单变量 CCF 定义衔接）。

**小例子（两列在同一时刻的相关系数，相当于 CCF 在 k=0）**  
仍用上面三行数据：第 1 列 x，第 2 列 y（把 x_2 看成 y_t）。

- 已算得：x 的样本方差（除数 n-1=2）为 1，y 的为 100，协方差为 10  
- 相关系数 rho_xy(0) = 10 / sqrt(1 * 100) = 1  

含义：这三行里 y 恰好是 x 的 10 倍，两列完全线性正相关，所以 k=0 时相关系数为 1。  
若你改成「x 涨 y 跌」的列，rho_xy(0) 会变成负数；若再算 rho_xy(1)，就要按 1.3 节把 y 错一行与 x 配对，看领先滞后（这里不再展开算式，避免与 1.3 重复堆公式）。

## 四、为什么叫「多维时间序列」而不只说「一张表」？
强调两点，和普通「只有行号没有先后逻辑」的表不同：

1. **时间是主轴**：行必须按时间顺序读，不是 i.i.d. 横截面样本；建模、预测、检验都依赖这条时间轴。
2. **列与列之间要联动分析**：不只单独看每一列，还要看列与列的动态关系（成交量、波动率是否领先价格等），这正是多维时序区别于「各列互不相干地当横截面」的地方。

---

## 五、多维模型基础：向量自回归（VAR）
一阶向量自回归 VAR(1) 可写为：
x_t = Phi * x_{t-1} + w_t

- 其中 Phi 是系数矩阵，w_t 是向量噪声项
- 矩阵形式可刻画多个序列之间的动态联动与相互影响

---

## 六、量化交易中的意义
1. 配对交易、多因子模型，本质都在处理多维时间序列关系。
2. CCF 可用于识别领先指标（如成交量、波动率对价格的领先效应）。
3. 是多变量时序建模、宏观因子联动分析、VAR 实务应用的基础。

## 七、Python 小例子：建表、均值向量、样本协方差
下面用三列模拟「价格、成交量、波动率」，便于你本地跑一遍，把「表」与「向量」对上号。

```python
import numpy as np
import pandas as pd

np.random.seed(0)
n = 5
# 模拟三列时序（仅演示结构，不是真实行情生成过程）
price = 100 + np.cumsum(np.random.randn(n))
volume = 5000 + 200 * np.random.randn(n)
vol = 0.02 + 0.005 * np.random.rand(n)

df = pd.DataFrame(
    {"t": np.arange(1, n + 1), "x_1_price": price, "x_2_volume": volume, "x_3_vol": vol}
)
X = df[["x_1_price", "x_2_volume", "x_3_vol"]]

# 均值向量：按列平均
mu_hat = X.mean()
print("数据表：")
print(df)
print("\n均值向量 mu_hat（每列样本均值）：")
print(mu_hat)

# 样本协方差矩阵（lag 0）：pandas 默认是列与列之间的协方差
Gamma0_hat = X.cov()
print("\n样本协方差矩阵 Gamma_hat(0)（列协方差）：")
print(Gamma0_hat)

# 两列之间的样本 CCF 可用 statsmodels 的 ccf；此处仅提示与 1.3 节公式一致
# from statsmodels.tsa.stattools import ccf
```

说明：
- 上面对 Gamma(0) 用的是**列与列同时刻**的样本协方差矩阵；滞后 k 的 Gamma(k) 需要对 (x_t - mu) 与 (x_{t+k} - mu) 做跨期配对，实现上可用自定义循环或专门的多变量时序包，思路与 1.3 节标量 gamma(k) 相同，只是把标量换成矩阵。
