# 1.2 时间序列统计模型

## 一、模型的核心思想
时间序列统计模型的本质：用数学解析式刻画观测数据背后的数据生成过程（DGP）。

通用表达：
x_t = f(I_{t-1}, w_t)

- I_{t-1}：t 时刻之前的历史信息（价格、成交量、波动率、滞后项等）
- w_t：当期随机噪声（新息、冲击）

建模拆解逻辑：
x_t = 信号_t + 噪声_t

- 信号项：趋势、周期、季节性、自相关、滞后传导（可建模、可预测）
- 噪声项：纯随机扰动、无规律部分（不可预测）

量化本质：尽可能剥离噪声、提取可交易信号，用于预测、择时、对冲和风控。

## 二、基础模型形式示例
### 1) 白噪声模型（White Noise）
   x_t = w_t

严格定义三条：
- E(w_t)=0
- Var(w_t)=sigma^2（常数）
- 任意非零滞后下自相关为 0（无记忆）

核心特点：
- 没有趋势、没有周期、没有自相关
- 完全随机，没有可交易信息

金融场景：
- 作为模型残差“是否合格”的基准
- 一个模型拟合后若残差接近白噪声，说明可解释信息提取较充分

### 2) 随机游走模型（Random Walk）
   x_t = x_{t-1} + w_t

含义：今天价格 = 昨天价格 + 随机新息。

关键性质：
- 无固定均值，方差随时间扩散
- 典型非平稳序列
- 长期难以精准点预测

金融场景：
- 弱有效市场下价格行为的经典基准
- 常用于解释“方向难预测、但波动结构可建模”的现象

### 3) 一阶自回归模型 AR(1)
   x_t = phi * x_{t-1} + w_t

参数理解：
- phi 是自回归系数，通常要求 -1 < phi < 1（平稳条件）
- 当前值由上一期线性驱动，再叠加当期噪声

系数直觉：
- phi 接近 1：强记忆、强惯性
- phi 接近 0：接近白噪声
- phi 的正负：决定更偏动量延续还是均值回归

模型地位：
- AR(p)、ARMA、ARIMA 等模型的基础原子结构

## 三、在量化交易中的意义
1. 所有量化策略都隐含某种时序模型假设：
   - 趋势策略依赖正自相关结构
   - 均值回归策略依赖负相关或稳定回归结构
   - 波动率交易依赖条件异方差结构

2. 对冲与调仓频率也受时序特征影响：
   - 序列记忆弱可降低调仓频率
   - 波动聚集强需要更高频动态风控

3. 回测有效性本质上取决于模型是否匹配真实 DGP：
   - 脱离时序结构的信号容易过拟合
   - 贴合 DGP 的信号更可能跨行情稳定

## 四、进阶模型谱系（承上启下）
在上述基础模型上，形成常用量化时序工具链：
- AR(p)：多阶自回归
- MA(q)：移动平均噪声结构
- ARMA(p,q)：AR + MA 混合
- ARIMA：加入差分处理非平稳序列
- GARCH 族：建模时变方差与波动聚集
- VAR：多变量联动建模

## 五、本节一句话总结
白噪声是“无信息基准”，随机游走是“价格基准”，AR(1)是“记忆基准”；后续预测、择时、波动率建模与对冲框架都由此展开。
