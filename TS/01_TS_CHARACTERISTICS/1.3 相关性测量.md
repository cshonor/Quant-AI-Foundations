# 1.3 相关性测量（纯新手·零门槛版）

## 先讲：到底什么叫【滞后】？
时间序列就是按时间排好的一串数字。比如：
x = [1, 2, 3, 4, 5]

### 什么叫滞后1（k=1）
就是今天和前一天配对，错开1个位置：
(1,2), (2,3), (3,4), (4,5)

### 什么叫滞后2（k=2）
就是今天和前两天配对，错开2个位置：
(1,3), (2,4), (3,5)

万能白话：
滞后 k = 往前数 k 天，做错位配对。

---

## 0) 三个必须记住的缩写
- ACF：同一序列，自己和自己错位比
- PACF：同一序列，去掉中间影响后再比
- CCF：两条不同序列，互相错位比

---

## 一、本节要解决的三个问题
1. 今天和过去几天到底有没有关系？
2. 建模该用 AR(1) 还是 AR(2)？
3. 看到的相关性是真信号还是噪声？

---

## 二、第一个工具：ACF（自相关函数）

### 1) 白话意思
同一串数据，自己和自己往前错位，看有没有关系。

### 2) 三个基础公式
先算均值：
x_bar = 总和 / 个数

再算错位协方差（滞后 k）：
gamma_hat(k) = (1/n) * 对 t 从 k+1 到 n 求和 [ (x_t - x_bar)(x_{t-k} - x_bar) ]

最后归一化得到 ACF：
rho_hat(k) = gamma_hat(k) / gamma_hat(0)

说明：
- gamma_hat(0) 就是样本方差
- rho_hat(k) 在 [-1, 1] 之间

### 3) 新手数字例题（手算）
样本：x = [1, 2, 3, 4, 5]

第一步：均值
x_bar = (1+2+3+4+5)/5 = 3

第二步：k=0
gamma_hat(0) = [(1-3)^2 + (2-3)^2 + (3-3)^2 + (4-3)^2 + (5-3)^2] / 5
            = (4 + 1 + 0 + 1 + 4) / 5
            = 10/5 = 2

第三步：k=1
配对：(1,2), (2,3), (3,4), (4,5)
gamma_hat(1) = [(1-3)(2-3) + (2-3)(3-3) + (3-3)(4-3) + (4-3)(5-3)] / 5
            = [2 + 0 + 0 + 2] / 5
            = 4/5 = 0.8

第四步：ACF(1)
rho_hat(1) = 0.8 / 2 = 0.4

结论：这组数据在滞后1下存在中等正相关。

### 4) ACF回答的问题
今天和过去几天有没有关系，先看 ACF。

## 三、第二个工具：PACF（偏自相关函数）

### 1) 白话意思
ACF 是总相关（含中间传递影响）。  
PACF 会把中间滞后项影响剥离，只看“纯直接相关”。

例如看 x_t 和 x_{t-2}：
- ACF 会把 x_{t-1} 的传递效应算进去
- PACF 会把 x_{t-1} 的影响剔除

### 2) 作用
PACF 主要用于给 AR 模型定阶：
- 在1阶后截尾 -> 常从 AR(1) 起步
- 在2阶后截尾 -> 常考虑 AR(2)

### 3) 计算提示
PACF 通常由软件通过 Yule-Walker 或回归法计算，新手先懂含义和用途。

---

## 四、第三个工具：CCF（互相关函数）

### 1) 白话意思
两条不同序列之间，在不同滞后下的相关关系。

例如：
- 价格 vs 成交量
- 收益率 vs 波动率
- 因子值 vs 未来收益

### 2) 公式结构
互协方差：
gamma_hat_xy(k) = (1/n) * 对 t 从 k+1 到 n 求和 [ (x_t - x_bar)(y_{t-k} - y_bar) ]

归一化后：
rho_hat_xy(k) = gamma_hat_xy(k) / sqrt(gamma_hat_x(0) * gamma_hat_y(0))

### 3) 作用
判断两个序列谁领先谁、谁滞后谁。

### 4) 最小手算例子（k=1）
设：
- x = [1, 2, 3, 4, 5]
- y = [0, 1, 2, 3, 4]

先算均值：
- x_bar = 3
- y_bar = 2

按定义（k=1）：
gamma_hat_xy(1) = (1/5) * 对 t=2..5 求和 [ (x_t-3)(y_{t-1}-2) ]

逐项：
- t=2: (2-3)*(0-2) = (-1)*(-2) = 2
- t=3: (3-3)*(1-2) = 0
- t=4: (4-3)*(2-2) = 0
- t=5: (5-3)*(3-2) = 2

所以：
gamma_hat_xy(1) = (2+0+0+2)/5 = 0.8

再算归一化分母：
- gamma_hat_x(0)=2
- gamma_hat_y(0)=2

CCF(1)：
rho_hat_xy(1) = 0.8 / sqrt(2*2) = 0.4

解释：
在这个例子里，k=1 时互相关为正，表示 y 的前一期与 x 的当期存在正向联动。

## 五、怎么区分“真信号”和“噪声”
ACF/PACF 图里常有置信区间虚线（约95%）：
- 落在虚线内：通常不显著，可能是噪声
- 超出虚线外：统计上更可能是有效相关

常配合 Ljung-Box 检验判断序列是否接近白噪声。

---

## 六、模型识别速用表
1. ACF拖尾 + PACF在 p 阶截尾 -> AR(p)
2. ACF在 q 阶截尾 + PACF拖尾 -> MA(q)
3. ACF拖尾 + PACF拖尾 -> ARMA(p,q)

---

## 七、和 1.5 节怎么衔接
1. 1.5 节先算样本自协方差
2. 再归一化得到样本 ACF
3. 再通过方法估计 PACF
4. 两序列交叉估计得到 CCF

---

## 八、最终速记（背这5句）
1. 滞后 = 往前错几天做配对
2. ACF = 自己和自己比，看总相关
3. PACF = 去掉中间影响，看纯直接相关
4. CCF = 两条不同序列比，看跨序列联动
5. 虚线内偏噪声，虚线外更可能是有效信号
