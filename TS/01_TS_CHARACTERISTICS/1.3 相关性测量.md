# 1.3 相关性测量

## 0. 先纠正缩写（高频口误）
正确三个缩写是：
1. ACF（自相关函数）
2. PACF（偏自相关函数）
3. CCF（互相关函数）

常见口误是把 ACF/PACF 说成 CFA、CF。复习时请始终用 ACF、PACF、CCF。

## 一、核心目标
这一节是从“理解模型”到“实际建模”的关键桥梁。

核心问题：
1. 上期和本期是否存在真实关系？
2. 模型应该选 AR(1)、AR(2) 还是 ARMA？
3. 观察到的相关性是信号还是噪声？

相关性测量就是一套量化尺子，用来：
- 判断序列是否有记忆（自相关）
- 判断不同序列是否联动（互相关）
- 给 AR/MA/ARMA 定阶
- 区分真信号与随机噪声

## 二、三大核心工具：ACF / PACF / CCF

### 1) 自相关函数 ACF（Auto-Correlation Function）

#### 1.1 定义（直观）
ACF 衡量“同一序列”和“其滞后 k 期版本”的相关程度。
例如看 x_t 与 x_{t-1}, x_{t-2}, ..., x_{t-k} 的相关系数。

关注对象：
- x_t 与 x_{t-1}, x_{t-2}, ..., x_{t-k}

一句话：自己跟自己比，看“总相关”（包含间接传递影响）。

#### 1.2 公式（掌握含义）
rho_k = gamma_k / gamma_0

- gamma_k：x_t 与 x_{t-k} 的协方差
- gamma_0：序列方差（用于归一化）
- 取值范围：-1 <= rho_k <= 1
  - rho_k = 1：完全正相关
  - rho_k = -1：完全负相关
  - rho_k = 0：无线性相关

关键补充：
1. ACF 的定义默认在弱平稳框架下使用；
2. ACF 是“总相关”，会包含中间滞后项的间接传递效应。

#### 1.3 ACF 图怎么读
- 衰减慢：记忆强、趋势强（随机游走会非常慢）
- 衰减快：记忆弱，通常更接近平稳结构
- 某阶后明显不显著：可视为“截尾”特征（MA 模型常见）

#### 1.4 量化应用
- 期货收益率 ACF 常较弱，但波动率序列 ACF 常缓慢衰减（波动聚集）
- 期权隐含波动率（IV）序列 ACF 常显著为正，可用于预测建模

### 2) 偏自相关函数 PACF（Partial Auto-Correlation Function）

#### 2.1 定义（直观）
PACF 是剔除中间滞后项影响后，x_t 与 x_{t-k} 的“直接相关”。

例如 x_t 与 x_{t-2} 的相关，可能是通过 x_{t-1} 传导出来的；
PACF 会把这种传导效应剥离，保留净关系。

一句话：自己跟自己比，但只看“纯直接相关”。

#### 2.2 核心作用：给 AR 模型定阶
- 若 PACF 在 p 阶后不显著（截尾），常选 AR(p)
- 例如 PACF 在1阶后截尾，常从 AR(1) 起步

#### 2.3 常见误区
ACF 是“总相关（含传导）”，PACF 是“净相关（剥离传导）”。
两者要结合看，不能只看其中一个。

### 3) 互相关函数 CCF（Cross-Correlation Function）

#### 3.1 定义
CCF 衡量两个不同序列在不同滞后下的相关关系。

例如：价格与成交量、标的收益率与期权 IV 的滞后联动关系。

一句话：A 和 B 两条不同序列之间的相关。

#### 3.2 量化应用
- 跨品种联动：看谁领先谁（如铜铝、股指与期指）
- 波动率交易：识别收益冲击与 IV 跳升的滞后关系
- 多因子建模：判断因子领先/滞后收益关系

## 三、模型识别万能对照（实操）
1. ACF 拖尾 + PACF 在 p 阶截尾 => AR(p)
2. ACF 在 q 阶截尾 + PACF 拖尾 => MA(q)
3. ACF 拖尾 + PACF 拖尾 => ARMA(p,q)

这套对照是建模初筛常用起点。

## 四、统计显著性：区分真相关和噪声

### 1) 置信区间
ACF/PACF 图上的虚线通常对应 95% 置信区间。
- 落在区间内：大概率不显著（可能是噪声）
- 超出区间：相关性统计上显著

### 2) Ljung-Box 白噪声检验
- 原假设：序列（或残差）是白噪声
- 若 p 值 < 0.05：拒绝原假设，说明仍有自相关结构

用途：
1. 建模前：判断序列是否存在可利用信息
2. 建模后：判断残差是否被“清洗”为白噪声

## 五、和前一节模型的直接关系
你前面学的 AR(1)、AR(p)、MA、ARMA，阶数并不是拍脑袋选。
核心依据就是本节的 ACF/PACF/CCF 与显著性检验结果。

例如：
- 若 PACF 在2阶后截尾，通常应考虑 AR(2) 而非 AR(1)
- 若 ACF 在1阶后截尾，常优先试 MA(1)

## 六、本节一句话总结
相关性测量就是给时间序列做“体检”：
- ACF 看整体记忆
- PACF 看直接滞后关系并用于 AR 定阶
- CCF 看跨序列联动

后续 AR/MA/ARMA/ARIMA 建模都应建立在这一步之上。

## 七、和 1.5 节的对应关系（联动复习）
1. 1.5 节先从样本自协方差出发，得到样本 ACF。
2. 在样本相关基础上，剔除中间滞后干扰，得到样本 PACF。
3. 两个不同序列做交叉相关估计，得到样本 CCF。

速记：
- ACF：自己 vs 自己，总相关
- PACF：自己 vs 自己，净相关
- CCF：A 序列 vs B 序列，跨序列相关

## 八、统计量怎么一步步算（手算版）

设样本序列为 x_1, x_2, ..., x_n，先算样本均值：
x_bar = (1/n) * sum_{t=1..n} x_t

### 0) 理论公式总览（弱平稳版）
序列方差（0阶自协方差）：
gamma_0 = Var(x_t) = E[(x_t - mu)^2]

k 阶自协方差：
gamma_k = Cov(x_t, x_{t-k}) = E[(x_t - mu)(x_{t-k} - mu)]

自相关系数：
rho_k = gamma_k / gamma_0

含义：
- gamma_0 衡量序列自身波动强度；
- gamma_k 衡量当前值与 k 期前的联动强度；
- rho_k 是归一化后可比较的相关强度。

### 1) 样本自协方差 gamma_hat(k)
给定滞后 k：
gamma_hat(k) = (1/n) * sum_{t=1..n-k} (x_t - x_bar) * (x_{t+k} - x_bar)

计算流程：
1. 先去均值：x_t - x_bar
2. 做错位配对：第 t 个和第 t+k 个
3. 两两相乘后求和
4. 最后除以 n

特例：
当 k=0 时，
gamma_hat(0) = (1/n) * sum_{t=1..n} (x_t - x_bar)^2
它就是样本方差（按 1/n 版本定义）。

### 2) 样本自相关 ACF：rho_hat(k)
rho_hat(k) = gamma_hat(k) / gamma_hat(0)

本质：把自协方差做归一化，得到范围通常在 [-1, 1] 的相关系数。

### 3) 样本互相关 CCF（两个序列 x_t, y_t）
先算互协方差：
gamma_hat_xy(k) = (1/n) * sum_{t=1..n-k} (x_t - x_bar) * (y_{t+k} - y_bar)

再归一化得到互相关：
rho_hat_xy(k) = gamma_hat_xy(k) / sqrt(gamma_hat_x(0) * gamma_hat_y(0))

这和普通相关系数结构一致，只是引入了“滞后 k”。

### 4) PACF 怎么算（原理版）
PACF 不是简单一条求和公式就能直接算出的，它的定义是：
“剥离中间滞后项后，x_t 与 x_{t-k} 的净相关”。

实操通常通过两种方式得到：
1. Yule-Walker 方程
2. 线性回归残差法（先把中间项解释掉，再算残差相关）

所以手算 PACF 一般不做全流程推导，建模时直接由软件计算。

### 5) 速记口诀
1. 先去均值
2. 同序列错位相乘求和 -> 自协方差
3. 除方差 -> ACF
4. 两序列错位相乘求和 -> 互协方差 -> CCF
5. 剥离中间传递影响 -> PACF

### 6) 极简手算示例（k=1）
样本：x = [1, 2, 3, 4, 5]，n=5

1. 均值：
x_bar = (1+2+3+4+5)/5 = 3

2. 计算 gamma_hat(1)：
配对为 (1,2), (2,3), (3,4), (4,5)
去均值后乘积：
(-2)*(-1) + (-1)*0 + 0*1 + 1*2 = 2 + 0 + 0 + 2 = 4
gamma_hat(1) = 4/5 = 0.8

3. 计算 gamma_hat(0)：
(-2)^2 + (-1)^2 + 0^2 + 1^2 + 2^2 = 10
gamma_hat(0) = 10/5 = 2

4. ACF(1)：
rho_hat(1) = 0.8 / 2 = 0.4

## 九、为什么这些统计量“可以算”（本质澄清）

### 1) 先定死一句话
时间序列本质上是“无穷多个随机变量按时间排队”：
X_1, X_2, X_3, ..., X_t, ...

每个时刻的 X_t 都是一个随机变量。  
所以你说“每个时间点本身是不同随机变量”，这个理解完全正确。

### 2) 核心疑问
既然每个 X_t 都不同，为什么还能写：
1. E(X_t)=mu（均值统一）
2. Var(X_t)=sigma^2（方差统一）
3. Cov(X_t, X_{t+k})=gamma(k)（只看间隔，不看位置）

答案：这不是天然成立，而是我们施加了“宽平稳（弱平稳）假设”。

### 3) 三条约束是“人为加上的建模条件”
宽平稳要求：
1. 所有时刻均值相同；
2. 所有时刻方差相同；
3. 协方差只由滞后间隔 k 决定，不由起点 t 决定。

也就是说：
- 不是所有序列都满足；
- 只有满足（或近似满足）宽平稳的序列，ACF/PACF 这些统计量才稳定可解释。

### 4) 为什么必须这样做
现实里我们通常只有一条历史路径，没有“平行世界重复实验”。
如果不加平稳性约束：
- 期望不稳定
- 协方差不稳定
- ACF/PACF 会随时间漂移
- 模型参数难以估计、难以复用

所以平稳假设的作用是：
让一条历史路径可以被当作可统计、可估计、可建模的数据来源。

### 5) 一句话锁定
不是时间序列天生满足这些性质；  
是为了能计算相关统计量并进行建模，我们主动给序列加上“宽平稳”这层结构约束。
