## 本章核心目标
掌握时间序列背景下的回归模型，学会用探索性分析和平滑方法处理数据。

## 关键概念
1. **时间序列回归**：经典线性回归在时序数据中的应用，注意自相关误差问题
2. **探索性数据分析（EDA）**：描述性统计、正态性检验、箱线图/QQ图
3. **平滑方法**：
   - 移动平均（MA）：简单移动平均、加权移动平均
   - 指数平滑（EWMA）：指数加权移动平均，给近期数据更高权重

## 量化应用场景
- 因子有效性检验：用未来收益对因子值做回归，看系数显著性
- 构建均线类因子（5日/20日均线）
- 平滑因子 IC 序列，去除噪声，观察长期趋势
- 用 EWMA 计算波动率，用于风险控制

## 实战练习
```python
# 1. 因子有效性回归检验
import statsmodels.api as sm
import pandas as pd

# 假设df包含因子值factor和未来收益future_ret
X = sm.add_constant(df['factor'])
model = sm.OLS(df['future_ret'], X).fit()
print(model.summary())

# 2. EWMA波动率计算
df['vol_ewma'] = df['ret'].ewm(span=20).std()
```
