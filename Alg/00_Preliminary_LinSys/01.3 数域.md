# 1.3 数域（补充有理数/实数域示例版）

## 为什么要引入数域

线性方程组的高斯消元、初等行变换、秩、解的存在唯一性判定，不依赖“实数”这一个具体集合。  
只要运算对象来自一个合法数域，整套方法与结论都成立。

## 数域的严格定义（纯文本版）

设数集 P（可看作复数集 C 的子集），若满足：

1. P 至少包含 0 和 1
2. 对任意 a,b 属于 P：a+b、a-b、a*b 仍属于 P
3. 对任意 a,b 属于 P 且 b != 0：a/b 仍属于 P

则称 P 为一个数域。

一句话：对加减乘除（除数非零）封闭。

## 三个基础数域与示例

### 1) 有理数域 Q

定义：所有 p/q（p,q 属于 Z, q != 0）组成的集合。

示例方程组：

(1/2)x_1 + (1/3)x_2 = 5/6  
x_1 + 2x_2 = 3

增广矩阵：

[ 1/2   1/3 | 5/6 ]
[ 1      2  |  3  ]

取消元系数 k = 1/(1/2) = 2，做 R2 <- R2 - 2R1：

[ 1/2   1/3 | 5/6 ]
[  0    4/3 | 4/3 ]

回代：

4/3 x_2 = 4/3 -> x_2 = 1  
代回第一式得 x_1 = 1

说明：全过程都在 Q 内完成。

---

### 2) 实数域 R

定义：所有有理数与无理数组成的集合。

示例方程组：

sqrt(2)x_1 + 3x_2 = 5  
x_1 - sqrt(3)x_2 = 2

增广矩阵：

[ sqrt(2)      3   | 5 ]
[   1      -sqrt(3)| 2 ]

取 k = 1/sqrt(2)，做 R2 <- R2 - kR1 后：

[ sqrt(2)               3           |        5         ]
[   0      -sqrt(3)-3/sqrt(2)       |   2-5/sqrt(2)    ]

再回代即可。

说明：无理数不会改变消元逻辑，只是运算表达更复杂。

---

### 3) 复数域 C

定义：所有 a+bi（a,b 属于 R，i^2 = -1）组成的集合。

示例方程组：

(1+i)x_1 + 2x_2 = 3  
3x_1 + (2-i)x_2 = 4

增广矩阵：

[ 1+i    2   | 3 ]
[  3   2-i   | 4 ]

选 k 使左下角消为 0：

3 - k(1+i) = 0 -> k = 3/(1+i)

共轭化简：

k = 3/(1+i)  
  = 3(1-i)/[(1+i)(1-i)]  
  = 3(1-i)/(1 - i^2)  
  = 3(1-i)/2

再做 R2 <- R2 - kR1，化成行阶梯后回代即可。

说明：步骤与实数域完全一致，仅多了复数运算细节。

---

## 核心对比表（纯文本）

Q：元素是分数；消元形式一致；主要是分数四则运算  
R：元素是实数（含无理数）；消元形式一致；主要是实数四则运算  
C：元素是复数；消元形式一致；主要是复数四则运算与共轭化简

结论：线性方程组的消元、秩与解判定是数域无关的结构性结论。

---

## 非数域示例（理解封闭性）

- Z（整数集）不是数域：除法不封闭（1/2 不在 Z）
- Q+（正有理数集）不是数域：减法不封闭（1-2 = -1 不在 Q+）
