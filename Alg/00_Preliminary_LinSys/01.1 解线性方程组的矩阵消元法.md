# 1.1 解线性方程组的矩阵消元法

## 核心思想

求解线性方程组时，先转化为增广矩阵，再用初等行变换等价化简为行阶梯形矩阵，最后自下而上回代。  
同一套流程不仅能求解，还能顺便判定是唯一解、无穷多解还是无解。

## 三种初等行变换（不改变解集）

1. 两行互换：ri <-> rj
2. 数乘行：ri <- k*ri（k != 0）
3. 行倍加：rj <- rj + k*ri

## 标准步骤

1. 写出增广矩阵 Abar = (A | b)
2. 逐列消元，化为行阶梯形矩阵
3. 比较 r(A) 与 r(Abar) 判定解的类型
4. 自下而上回代，得到未知数

---

## 完整例题（实数域）

方程组：

x_1 + 2x_2 = 3
3x_1 + 4x_2 = 7

步骤1：写出增广矩阵

[ 1  2 | 3 ]
[ 3  4 | 7 ]

步骤2：做行变换 R2 <- R2 - 3R1

[ 1   2 |  3 ]
[ 0  -2 | -2 ]

此时已是行阶梯形。

步骤3：回代

-2x_2 = -2 -> x_2 = 1
x_1 + 2 = 3 -> x_1 = 1

结论：唯一解 `x_1=1, x_2=1`。

---

## 衔接后续：秩判定三结论

设未知数个数为 n：

1. r(A) = r(Abar) = n  -> 唯一解
2. r(A) = r(Abar) < n  -> 无穷多解
3. r(A) < r(Abar)      -> 无解
