# 1.2 线性方程组解的情况及判别准则

## 核心判定逻辑

将增广矩阵化为行阶梯形后，按以下顺序判断：

1. **先判无解**  
   若出现行 `[\ 0\ \cdots\ 0\ |\ c\ ] (c \neq 0)`，即 `0=c`，则无解。
2. **有解充要条件**  
   系数矩阵秩与增广矩阵秩相等：`r(A)=r(\bar{A})`。

## 有解后的分类

设未知数个数为 `n`，秩为 `r`：

- `r=n`：无自由变量，唯一解
- `r<n`：有 `n-r` 个自由变量，无穷多解

其中，自由变量个数为：

n - r

## 例子（含自由变量）

x_1 + x_2 + x_3 + x_4 = 5
x_1 + 2x_2 + 3x_3 + 4x_4 = 7
2x_1 + 3x_2 + 4x_3 + 5x_4 = 12

对应增广矩阵为：

[ 1  1  1  1 |  5 ]
[ 1  2  3  4 |  7 ]
[ 2  3  4  5 | 12 ]

化为行阶梯矩阵的一组过程：

1) R2 <- R2 - R1
2) R3 <- R3 - 2R1

得到：

[ 1  1  1  1 | 5 ]
[ 0  1  2  3 | 2 ]
[ 0  1  2  3 | 2 ]

3) R3 <- R3 - R2

得到行阶梯矩阵：

[ 1  1  1  1 | 5 ]
[ 0  1  2  3 | 2 ]
[ 0  0  0  0 | 0 ]

可见主元行数为 2，所以秩 r = 2。

对该方程组做行化简后，只有两行彼此独立的非零方程（第三行可由前两行线性表示），因此系数矩阵秩 r=2；同时未知数为 x_1, x_2, x_3, x_4 共 4 个，即 n=4。于是可据此判断解空间维数与自由变量个数：

- 自由变量个数 `=4-2=2`
- 可设 `x_3=a, x_4=b`
- `x_1, x_2` 可表示成 `a, b` 的线性表达式

下面给出一组可直接复用的计算过程：

1) 用第 1 个方程消去后两式中的 x_1

- (第2式 - 第1式):  
  x_2 + 2x_3 + 3x_4 = 2
- (第3式 - 2*第1式):  
  x_2 + 2x_3 + 3x_4 = 2

可以看到后两式完全相同，所以独立有效方程只有 2 个，故秩 r = 2。

2) 设自由变量

设 x_3 = a, x_4 = b。

3) 先解主元变量 x_2

由 x_2 + 2x_3 + 3x_4 = 2，代入得：

x_2 = 2 - 2a - 3b

4) 再解 x_1

由第 1 式 x_1 + x_2 + x_3 + x_4 = 5，代入得：

x_1 = 5 - x_2 - x_3 - x_4
    = 5 - (2 - 2a - 3b) - a - b
    = 3 + a + 2b

最终通解可写为：

x_1 = 3 + a + 2b
x_2 = 2 - 2a - 3b
x_3 = a
x_4 = b

其中 a, b 可任意取值。

结论：该方程组有无穷多解，由两个自由参数控制。
