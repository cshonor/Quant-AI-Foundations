# 2.5 克拉默（Cramer）法则

## 适用前提

n 个方程、n 个未知数的非齐次线性方程组：

A x = b

设 D = det(A)。

## 结论

1. 若 D != 0，则方程组有且仅有唯一解
2. 各未知数可写为：

x1 = D1/D, x2 = D2/D, ..., xn = Dn/D

其中 Di 表示把 A 的第 i 列替换为常数列 b 后得到的行列式。

## 齐次方程组推论

对 A x = 0：

- D != 0 -> 只有零解
- D = 0 -> 存在非零解（有非平凡解）

## 优缺点

- 优点：理论清晰，给出显式公式
- 缺点：高阶计算量大，实际数值计算中不如消元法高效
