# 2.6 拉普拉斯展开（按 k 行/列）与分块行列式

## k 阶子式

在 n 阶行列式中任选 k 行、k 列，交叉位置元素构成的 k 阶行列式称为 k 阶子式。

## 拉普拉斯展开定理（思想）

固定一组 k 行（或 k 列）：

- 这些位置产生的所有 k 阶子式
- 分别乘对应代数余子式
- 全部求和

结果等于原 n 阶行列式。

## 最常用结论：分块三角型

若

| A  C |
| 0  B |

其中 A、B 为方阵，0 为零块，则

det = det(A) * det(B)

同理，

| A  0 |
| C  B |

也有 det = det(A) * det(B)。

## 用途

这条结论在分块矩阵计算中非常高频，常用于快速求高阶行列式。
