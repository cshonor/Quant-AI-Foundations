# 2.4 行列式按一行（列）展开

## 余子式与代数余子式

- 余子式 Mij：删除第 i 行和第 j 列后得到的 (n-1) 阶行列式
- 代数余子式 Aij：Aij = (-1)^(i+j) * Mij

## 展开定理

按第 i 行展开：

det(A) = ai1*Ai1 + ai2*Ai2 + ... + ain*Ain

按第 j 列展开：

det(A) = a1j*A1j + a2j*A2j + ... + anj*Anj

## 正交性质（常用）

1. 同一行元素与“另一行”的代数余子式乘积和为 0
2. 同一列元素与“另一列”的代数余子式乘积和为 0

## 计算技巧

优先选择 0 最多的行或列展开，可大幅减少计算量。

## 例子

D =
|2 0 0|
|1 3 4|
|5 6 7|

按第一行展开：

D = 2*A11 + 0 + 0  
= 2 * |3 4; 6 7|  
= 2 * (21-24)  
= -6
