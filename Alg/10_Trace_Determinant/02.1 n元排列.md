# 2.1 n 元排列

## 基本定义

- n 元排列：由 1,2,...,n 这 n 个互不重复的数组成的有序数组，记作 j1j2...jn
- 排列总数：n!
- 标准排列：123...n

## 逆序与逆序数

- 逆序：在排列中，若大的数在前、小的数在后，则构成 1 个逆序
- 逆序数 tau：排列里所有逆序的总个数

计算方法：从左到右统计每个元素后面比它小的元素个数，再求和。

例：32514

- 3 后面比它小的有 2、1 -> 2
- 2 后面比它小的有 1 -> 1
- 5 后面比它小的有 1、4 -> 2
- 1 后面 -> 0
- 4 后面 -> 0

所以 tau(32514) = 2+1+2+0+0 = 5

## 奇排列与偶排列

- tau 为奇数：奇排列
- tau 为偶数：偶排列

上例 tau=5，所以是奇排列。

## 核心结论

1. 一次对换（交换任意两个数）会使排列奇偶性翻转
2. n>=2 时，奇排列个数 = 偶排列个数 = n!/2
3. 任意排列都可经若干次对换变为标准排列；对换次数奇偶性与该排列奇偶性一致

## 本节作用

这是 n 阶行列式定义的前置基础。  
行列式本质是“对所有排列按奇偶带符号求和”。
