# 2.2 n 阶行列式的定义

## 二阶、三阶回顾

二阶行列式：

|a11 a12|
|a21 a22| = a11*a22 - a12*a21

三阶行列式可用“主对角线正项 - 副对角线负项”记忆（仅适用于 3 阶）。

## n 阶行列式定义（纯文本）

设 A=(aij) 为 n 阶方阵。  
det(A) 的定义是：

1. 遍历列下标的全部 n! 个排列 j1...jn
2. 每一项取 a1j1 * a2j2 * ... * anjn
3. 若排列逆序数 tau 为偶数取正号，奇数取负号
4. 全部项求和

可写作：

det(A) = sum[ (-1)^tau(j1...jn) * a1j1*a2j2*...*anjn ]

## 基础结论

1. 上三角行列式 = 主对角线元素乘积
2. 下三角行列式 = 主对角线元素乘积
3. 对角行列式 = 主对角线元素乘积
