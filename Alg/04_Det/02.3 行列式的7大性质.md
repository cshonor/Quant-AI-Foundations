# 2.3 行列式的 7 大核心性质

以下性质在 Q、R、C 等数域中都成立。

## 性质 1：转置不改变行列式

det(A^T) = det(A)

行列地位对称。

## 性质 2：某一行有公因子可提出

若第 i 行都乘 k，则行列式整体乘 k。  
推论：某一行全为 0，则行列式为 0。

## 性质 3：交换两行，行列式变号

交换任意两行后，det 变为相反数。  
推论：若两行相同，则 det=0。

## 性质 4：两行成比例，行列式为 0

若某行 = k * 另一行，则 det=0。

## 性质 5：某一行可按加法拆分

若第 i 行 = u+v（逐元素），则原行列式 = 两个行列式之和。

## 性质 6：倍加行变换不改变行列式

执行 rj <- rj + k*ri，det 不变。  
这是“消元法求行列式”的核心。

## 性质 7：整矩阵数乘

对 n 阶矩阵：det(kA)=k^n * det(A)

注意是 k^n，不是 k。

## 计算建议

优先用“倍加行变换”把矩阵化成上三角，再取对角线乘积求值。
