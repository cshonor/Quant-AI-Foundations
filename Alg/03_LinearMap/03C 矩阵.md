# 矩阵
## 1. 矩阵与线性映射的一一对应
选定 V 的一组基、W 的一组基之后：
- 任意一个线性映射 T: V -> W，都可以唯一对应一个矩阵。
- 任意一个矩阵，也唯一确定一个线性映射。

## 2. 坐标表示
向量在基下可以写成坐标列向量，此时线性映射作用等价于**矩阵左乘列向量**：
[T(v)]在目标基下的坐标 = M_T 乘以 [v]在原基下的坐标列向量。

## 3. 矩阵运算的线性意义
1. 矩阵加法 = 线性映射加法
2. 矩阵数乘 = 线性映射数乘
3. 矩阵乘法 = 线性映射复合（先后执行两次变换）

## 4. 核心结论
矩阵只是线性映射在特定坐标系下的数值载体；线性映射是本质，矩阵只是表象。
