# 对偶空间
## 1. 对偶空间定义
向量空间 V 的对偶空间 V' 定义为：
V' = L(V, F)。
也就是：所有从 V 映到数域 F 的线性函数全体，本身也构成向量空间。

## 2. 对偶基
若 (v1, ..., vn) 是 V 的一组基，则可以唯一构造出 V' 的一组对偶基 (phi1, ..., phin)，满足：
phi_i(v_j) = delta_ij，即当 i = j 时取 1，当 i 不等于 j 时取 0。

## 3. 核心维数结论
有限维空间下：dim(V') = dim(V)。

## 4. 补充说明
对偶空间属于线性代数进阶抽象内容，入门阶段、侧重算子与应用的学习，可以延后处理。
