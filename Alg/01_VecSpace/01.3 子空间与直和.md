# 1.C 子空间与直和（通俗完整版）

先说两个常见疑问：
1. 线性空间 = 向量空间（同一个概念，两种叫法）。
2. R^3 就是三维欧式空间，也就是常见的 xyz 坐标空间。

## 1) 子空间判定
在大空间 V 里取一个子集 U。  
U 是子空间，当且仅当满足三条：
1. 包含零向量 0
2. U 里任意两个向量相加，结果还在 U
3. U 里任意向量乘任意标量，结果还在 U

人话：在 U 里面怎么“加”和“乘”都不会跑出去，而且一定经过原点。

## 2) R^3 里的直观例子
- 过原点的直线：是子空间
- 过原点的平面：是子空间
- 不过原点的直线或平面：不是子空间

## 3) 子空间的和
给两个子空间 U1、U2，定义：
U1 + U2 = { u1 + u2 | u1 属于 U1, u2 属于 U2 }

意思是：把“U1 里任意向量”和“U2 里任意向量”相加，所有结果放在一起。

## 4) 直和（重点）
先记一句话：  
直和 = 两个子空间只在零向量处相交。

### 4.1 普通的和（不一定是直和）
如果 U1 和 U2 有较多重叠，那么同一个向量可能有多种拆法：
alpha = u1 + u2

这时通常不是直和。

### 4.2 直和（干净的和）
当 U1 和 U2 只共享零向量时：
U1 交 U2 = {0}

这时每个向量的分解方式是唯一的。

### 4.3 最直白理解
1. 普通和：有重叠，分解可能不唯一
2. 直和：只在 0 相交，分解唯一

### 4.4 R^3 例子
- U1 = x 轴
- U2 = y 轴

两者公共部分只有 (0,0,0)。  
在它们张成的平面内，每个向量都能且只能拆成：
x 轴向量 + y 轴向量

所以这是直和。

## 5) 直和最简判断
1. 交集只有零向量
2. 向量分解只有一种方式

满足这两条，就可以判定为直和。
