# 线性代数
教材：这才是线性代数

归属仓库：机器学习仓库
配套学习：凸优化、实变函数、机器学习

本课程为高维数学、优化理论、量化学习的底层基础。
学习路线以线性代数核心体系为主，强调空间、变换与度量结构。

## 章节学习内容（按学习路线）
### 第1章 向量与线性方程组
- 向量、线性组合、张成
- 线性方程组与消元法
- 解空间与几何直观

### 第2章 矩阵与线性映射
- 矩阵运算与基本性质
- 线性映射及其矩阵表示
- 基变换与坐标变换

### 第3章 行列式
- 行列式定义与性质
- 代数余子式与展开公式
- 行列式在可逆性判定中的作用

### 第4章 向量空间
- 子空间、基与维数
- 线性相关与无关
- 秩-零度关系

### 第5章 线性变换进阶
- 像空间与核空间
- 变换复合与逆变换
- 不变子空间初步

### 第6章 特征值与特征向量
- 特征值、特征向量定义
- 特征多项式与代数重数
- 对角化条件与方法

### 第7章 内积空间与正交
- 内积、范数与距离
- 正交基与 Gram-Schmidt 正交化
- 正交投影与最小二乘

### 第8章 二次型与矩阵分解
- 二次型及标准化
- 实对称矩阵的谱分解
- 常见分解（LU/QR/SVD）与应用直觉

### 第9章 线性代数在机器学习中的连接
- 梯度下降与矩阵微分基础
- PCA 与低秩近似
- 线性代数在优化与建模中的核心作用
