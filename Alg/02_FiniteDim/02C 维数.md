# 维数
## 1. 维数的严格定义
有限维向量空间 V 的维数，记作 dim(V)，定义为：V 的任意一组基所含向量的个数。

- 零空间 {0} 的维数规定为 0；
- R^n 的维数是 n（标准基有 n 个向量）。

## 2. 维数不变性定理
核心基石结论：同一个有限维向量空间，所有基包含的向量数量完全相等。  
维数是空间本身的固有属性，和基的选取方式无关。

证明要点（简述）：
- 由 Steinitz 替换引理：若 A、B 都是 V 的基，则 |A| <= |B| 且 |B| <= |A|；
- 因此 |A| = |B|，基向量个数唯一。

几何直观：
- 二维平面不管怎么换基，始终只能有 2 个独立方向；
- 不可能出现 3 个彼此线性无关且仍留在同一平面内的方向。

## 3. 子空间维数性质
1. 若 U 是 V 的子空间，则 dim(U) <= dim(V)。
2. 若 U 是 V 的子空间，且 dim(U) = dim(V)，则必然有 U = V。

例子：
- 在 R^3 中，任一过原点平面的维数是 2，小于 3；
- 若某子空间在 R^3 中维数等于 3，则它只能是整个 R^3。

## 4. 维数判定准则
设 V 为 n 维向量空间：
1. 任意**n个线性无关向量**，自动构成一组基、必然张成全空间。
2. 任意**n个可以张成全空间的向量**，必然线性无关、自动成为一组基。

秒杀结论：
- 向量个数 > 维数 => 必线性相关；
- 向量个数 < 维数 => 必不能张成全空间。

## 5. 维数公式
对任意两个子空间 U1, U2，有维数恒等式：
dim(U1 + U2) + dim(U1 交 U2) = dim(U1) + dim(U2)。

几何直观：
- U1 + U2 是“合并后的空间”；
- U1 交 U2 是“重叠部分”；
- 公式本质是：合并维数 = 两者维数和 - 重叠维数。

例子（R^3）：
- U1 = xy 平面，dim(U1)=2
- U2 = xz 平面，dim(U2)=2
- U1 + U2 = R^3，dim(U1+U2)=3
- U1 交 U2 = x 轴，dim(U1 交 U2)=1
- 验证：3 + 1 = 2 + 2

## 6. 一句话口诀（考场速记）
- 维数 = 基的向量数
- 基长不变，维数固有
- 子空间维数不增，等维必相等
- n 维空间：n 个无关即成基，n 个张成也成基
- 和维 + 交维 = 维数和
