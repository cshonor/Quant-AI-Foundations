# 张成空间与线性无关
## 1. 张成空间（Span）
### 1.1 严格数学定义
设 V 是数域 F 上的向量空间，给定向量组 v1, v2, ..., vm 属于 V，该向量组的张成空间定义为：
span(v1, v2, ..., vm) = { lambda1*v1 + lambda2*v2 + ... + lambdam*vm | lambda1, ..., lambdam 属于 F }
也就是该组向量一切线性组合的全体集合。

### 1.2 核心性质
1. span(v1, ..., vm) 一定是 V 的线性子空间，对向量加法、数乘运算封闭。
2. span(v1, ..., vm) 是包含向量组 {v1, ..., vm} 的最小子空间：任何包含这组向量的子空间，必然包含它们的全部张成空间。
3. 如果 S 是向量空间的任意子集，则 span(S) 是由 S 生成的子空间。

### 1.3 空间张成
- 若 span(v1, ..., vm) = V，则称向量组 {v1, ..., vm} 张成整个空间 V。
- 张成，描述向量组“能覆盖整个空间、能表示空间中任意向量”的能力。

### 1.4 关键疑问：张成是否要求线性无关
结论：不要求。  
任何向量组都可以定义张成空间，不论是否线性无关。

也就是说，下列情况都照样可以谈“张成”：
- 向量组线性相关
- 向量组里有重复向量
- 向量组里含零向量

张成只关心“能组合出哪些向量”，不关心“是否有冗余”。

### 1.5 例子：线性相关也能张成
在 R^3 中取向量组：
- a = (1,0,0)
- b = (2,0,0)

这组向量线性相关，但它们的所有线性组合仍构成一个子空间：
span(a,b) = x 轴。

所以“线性相关”不会阻止“张成空间”存在。

## 2. 线性无关
### 2.1 严格定义
向量组 v1, v2, ..., vm 属于 V 称为线性无关，当且仅当向量方程
lambda1*v1 + lambda2*v2 + ... + lambdam*vm = 0
仅有零解：lambda1 = lambda2 = ... = lambdam = 0。

### 2.2 等价理解
- 线性无关：向量组中，**没有任何一个向量，可以写成其余向量的线性组合**。
- 向量之间互相独立，不存在冗余、不存在依赖关系。

## 3. 线性相关
### 3.1 严格定义
向量组 v1, v2, ..., vm 属于 V 称为线性相关，当且仅当存在不全为 0 的系数 lambda_i 属于 F，使得
lambda1*v1 + lambda2*v2 + ... + lambdam*vm = 0。

### 3.2 核心等价结论
线性相关 当且仅当 向量组中至少有一个向量可以被其余向量线性表示。

### 3.3 基础判定结论
先回顾定义：
- 线性相关：存在不全为 0 的系数 k1, ..., km，使 k1*a1 + ... + km*am = 0。
- 线性无关：只有 k1 = ... = km = 0 才能使线性组合等于 0。

#### 结论1：包含零向量的向量组一定线性相关
- 通俗解释：只要队伍里有 0 向量，就能立刻构造出“非全零系数但和为 0”。
- 标准证明：若 a1 = 0，取 k1 = 1，k2 = ... = km = 0，则
  k1*a1 + ... + km*am = 1*0 + 0 + ... + 0 = 0，
  且系数不全为 0，所以线性相关。
- 易错点：这里的“非全零系数”只要有一个不为 0 即可，不要求全都不为 0。

#### 结论2：线性无关向量组的任意子集仍线性无关
- 通俗解释：整体没有冗余，删掉一些向量后不会凭空产生冗余。
- 标准证明（逆否命题）：若子集 B 线性相关，则 B 中存在不全为 0 的系数使组合为 0。
  把全集 A 中不在 B 的向量系数补 0，就得到 A 的一组不全为 0 的系数也使组合为 0，
  所以 A 线性相关。故其逆否成立：A 线性无关 => 任意子集线性无关。
- 易错点：不要把方向写反。正确是“无关向下传递”，不是“无关向上扩张”。

#### 结论3：线性相关向量组的任意超集仍线性相关
- 通俗解释：已经有冗余了，再加新向量不会把冗余消掉。
- 标准证明：设 A 线性相关，则 A 中存在不全为 0 的系数使组合为 0。
  对超集 B，把 B\A 中新增向量系数全部取 0，原关系仍成立，且系数仍不全为 0，
  故 B 线性相关。
- 易错点：新增向量“可以无关”，但超集整体仍会保留原来的相关关系。

#### 结论4：n 维空间中向量个数大于 n 必线性相关
- 通俗解释：空间最多容纳 n 个彼此独立方向，再多一定有冗余。
- 标准证明（方程组观点）：设 a1, ..., am 属于 R^n，且 m > n。
  考察方程 k1*a1 + ... + km*am = 0，可写成 A(n×m)k = 0。
  未知数 m 多于方程数 n，齐次方程组必有非零解，所以向量组线性相关。
- 易错点：该结论是“个数 > 维数 => 必相关”，反过来“个数 <= 维数 => 必无关”是错的。

### 3.4 考场口诀
1. 有零必相关
2. 无关子集仍无关
3. 相关超集仍相关
4. 个数超维必相关

## 4. 本节核心定理
1. **线性相关引理**  
若向量组 (v1, ..., vm) 线性相关，则必存在某个 j 属于 {2, ..., m}，满足：
- vj 属于 span(v1, ..., v(j-1))
- 删掉 vj 之后，剩余向量张成的空间保持不变。

## 5. 一句话区分
- 张成：只看能否组合出目标向量，相关无关都可以。
- 基：必须线性无关，同时还能张成目标空间。
