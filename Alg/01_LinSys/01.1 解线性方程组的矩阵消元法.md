# 1.1 解线性方程组的矩阵消元法

## 核心思想

面对线性方程组，不用反复代入，先写成增广矩阵，再通过初等行变换把它化成行阶梯形，最后回代求解。

## 三种初等行变换（不改变解集）

1. 交换任意两行
2. 某一行乘以非零常数
3. 把某一行的 `k` 倍加到另一行

## 标准步骤

1. 写出增广矩阵（系数矩阵 + 常数列）
2. 逐步消元，化为行阶梯形
3. 从底向上回代，得到未知数

## 实数域示例

方程组：

x_1 + 2x_2 = 3
3x_1 + 4x_2 = 7

增广矩阵：

[ 1  2 | 3 ]
[ 3  4 | 7 ]

做变换 `R2 <- R2 - 3R1`：

[ 1   2 |  3 ]
[ 0  -2 | -2 ]

回代得：

-2x_2 = -2 -> x_2 = 1
x_1 + 2 = 3 -> x_1 = 1

结论：唯一解 `x_1=1, x_2=1`。
