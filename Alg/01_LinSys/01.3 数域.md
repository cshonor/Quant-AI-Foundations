# 1.3 数域

## 为什么要讲数域

在线性方程组理论里，消元、秩、解的判定并不只对实数成立，而是在任意“合法数域”中都成立。

## 什么是数域

数集 `P` 若满足：对任意 `a,b` 属于 `P`（且 `b != 0`），`a+b`、`a-b`、`ab`、`a/b` 仍在 `P` 中，则称 `P` 为数域。

## 常见数域

1. 有理数域 `Q`
2. 实数域 `R`
3. 复数域 `C`

## 复数域上的消元

高斯消元步骤与实数域完全一致，唯一差别是复数运算规则：

- `i^2=-1`
- 复数除法常用共轭化简分母

## 复数域示例（过程模板）

(1+i)x_1 + 2x_2 = 3
3x_1 + (2-i)x_2 = 4

增广矩阵：

[ 1+i    2   | 3 ]
[  3   2-i   | 4 ]

可取消元系数：

k 的作用：把第 2 行左下角元素消成 0，所以取

k = 3/(1+i)

为什么要这样取：

- 第 1 行第 1 列是 (1+i)
- 第 2 行第 1 列是 3
- 做 R2 <- R2 - k*R1 时，希望 3 - k(1+i) = 0
- 解这个方程得到 k = 3/(1+i)

复数除法化简（分子分母同乘分母共轭）：

k = 3/(1+i)
  = 3(1-i)/[(1+i)(1-i)]
  = 3(1-i)/(1 - i^2)
  = 3(1-i)/2

再做 `R2 <- R2 - kR1` 消去左下元素，化为行阶梯后回代即可。

结论：线性方程组的消元与秩判定是“数域无关”的结构性结论。
