# 第 1 章笔记：什么是机器学习

## 本章笔记整理

### 精简版

- 人类决策有两类：规则推理、经验判断。  
- 机器学习对应“经验判断”，其中经验在计算机里就是数据。  
- 当计算机通过数据学习并决策时，就是在进行机器学习。  
- 对复杂任务（如图像识别），硬编码规则难度高，数据驱动方法更有效。  

### 详细版

#### 核心概念

- **经验 = 数据**：机器学习不是人工穷举规则，而是让模型从数据中归纳规律。  
- **范式变化**：从“程序 + 数据 -> 结果”逐步转向“数据 + 学习算法 -> 模型 -> 结果”。  

#### 与传统编程对比

- 传统编程：人先定义规则，机器执行规则。  
- 机器学习：人提供样本，模型自动学习决策边界。  
- 当任务特征难以完全形式化时（如视觉、语音、自然语言），机器学习通常更可行。  

#### 示例理解（苹果识别）

- 手写规则法：需定义颜色、形状、大小、纹理、光照变化等大量条件，容易遗漏。  
- 数据学习法：通过大量标注样本学习“苹果”特征分布，泛化到新样本。  

#### 一句话总结

> 机器学习的本质是让计算机从数据中积累经验，并用这些经验进行预测和决策。

### 术语补充：模型和算法

#### 核心定义

- **模型（model）**：一组用于表示数据规律的规则，主要用于预测。  
- **算法（algorithm）**：为解决问题而执行的一组步骤，在机器学习里主要用于构建模型。  

#### 关系与区别

- 模型是“学出来的结果”，用于对新数据做判断或预测。  
- 算法是“学习的方法”，决定如何从数据中得到模型。  
- 可记为：**算法训练模型，模型负责预测**。  

#### 书中这一节想表达的重点

- 机器学习先建立术语体系，避免后续概念混淆。  
- “模型”和“算法”常被混用，但职责不同：一个偏结果，一个偏过程。  
- 实践中应区分两件事：  
  1) 我在用什么算法训练；  
  2) 我最终得到的模型表现如何。  

#### 一句话总结

> 模型是对数据规律的可用表达，算法是产生这种表达的步骤与方法。
