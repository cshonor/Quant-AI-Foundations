# RFA：实变函数与泛函分析

`RFA`（Real and Functional Analysis）用于系统整理实变函数与泛函分析的学习路径。

这两部分内容是凸优化、机器学习、量化研究的重要底层数学基础：

- 实变函数：提供测度、可测函数、Lebesgue 积分等现代分析工具，帮助理解概率与连续优化中的严谨表达。
- 泛函分析：提供赋范空间、Hilbert 空间、线性算子与谱理论框架，支撑优化算法、学习理论与算子方法。

本目录仅建立结构，不放置具体笔记文件，便于后续按主题逐步补充内容。
