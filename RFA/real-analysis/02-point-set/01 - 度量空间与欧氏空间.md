# 实变函数 第二章 第一节：度量空间与欧氏空间

## 1. 什么是度量空间

给定集合 `X`，再定义一个距离函数 `d(x, y)`。若满足以下三条公理，则 `(X, d)` 称为度量空间：

- 非负性与同一性：`d(x, y) >= 0`，且 `d(x, y) = 0` 当且仅当 `x = y`
- 对称性：`d(x, y) = d(y, x)`
- 三角不等式：`d(x, z) <= d(x, y) + d(y, z)`

只要是“集合 + 合法距离”，就是度量空间。

## 2. 什么是 n 维欧氏空间

`R^n` 是 n 维实向量（点）组成的集合。  
当 `R^n` 配上欧氏距离时，得到欧氏空间：

`d2(x, y) = sqrt(sum((xi - yi)^2))`

即：逐维相减、平方求和、再开根号。

## 3. 关键问题：换距离后还是不是欧氏空间？

在 `R^n` 上把距离改为其他形式（如 `L1`、`Lp`），如果仍满足度量三公理，则：

- 仍然是度量空间
- 但不再是欧氏空间（欧氏空间特指欧氏距离）

## 4. 核心区别（高频考点）

- 度量空间：范围更大，只要求距离合法
- 欧氏空间：度量空间的特例，使用固定的欧氏距离

## 5. 本节两个基础概念

### 邻域

点 `x0` 的 `delta` 邻域：

`N_delta(x0) = {x in X | d(x, x0) < delta}`

后续开集、内点、边界等概念都建立在邻域上。

### 点列极限

`xn -> x` 当且仅当 `d(xn, x) -> 0`。

## 一句话总结

度量空间是“集合 + 合法距离”；欧氏空间是其中使用欧氏距离的一类特例。  
在 `R^n` 中更换为其他合法距离后，仍是度量空间，但通常不再称为欧氏空间。
