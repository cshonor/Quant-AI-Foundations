# 第五节 康托尔三分集

## 一、康托尔集的构造

1. 初始集合：`C0 = [0,1]`；
2. 第 1 步：去掉中间开区间 `(1/3, 2/3)`，得  
   `C1 = [0,1/3] ∪ [2/3,1]`；
3. 第 2 步：对 `C1` 中每个闭区间再去掉其中间三分之一开区间，得到 `C2`；
4. 无限重复上述过程；
5. 康托尔集定义为：

`C = ∩(n=0 to inf) Cn`

## 二、康托尔集的基本性质

1. 是闭集：每个 `Cn` 闭，闭集可数交仍闭；
2. 无孤立点：每一点附近总有其他康托尔点；
3. 因而是完备集（完全集）：闭且无孤立点；
4. 无内点：不含任何正长度开区间；
5. 勒贝格测度为 0：被挖去长度总和为 1。

## 三、最反直觉的特点

1. 看起来“几乎被挖空”，长度为 0；
2. 但基数等于连续统，与 `R` 同阶，是不可数集。

## 四、学习意义

1. 是“闭且无内点但不可数”的经典模型；
2. 打破“长度为 0 就点很少”的直觉；
3. 是后续测度论、奇异函数的重要基础例子。
