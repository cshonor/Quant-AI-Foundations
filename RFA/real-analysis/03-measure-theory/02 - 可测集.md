# 可测集

## 阅读导航

- 先看：`可测性终极速记` + `高频混淆对照表`
- 主线：第 1~3 节（定义、核心性质、测度连续性）
- 进阶：第 8~9 节（等价证明与概念澄清）

---

## 0. 可测性终极速记（先看这8行）

1. 外测度 m^*(E)：从外覆盖，取最小代价（下确界）  
2. 内测度 m_*(E)：从内逼近，取最大代价（上确界）  
3. 永远有：m_*(E) <= m^*(E)  
4. 可测集判定：m_*(E) = m^*(E)  
5. 共同数值记为 m(E)，称为勒贝格测度  
6. 常见规整集合通常内外相等（可测）  
7. 不可测集合会出现严格不等：m_*(E) < m^*(E)  
8. 一句话：外量和内量能对上，就可测

---

## 0.1 高频混淆对照表（可数/有界/可测）

这三组概念彼此独立，最容易混淆：

| 概念 | 它回答的问题 | 典型判断标准 |
|---|---|---|
| 可数 / 不可数 | 元素“多少” | 能否与自然数建立一一对应（可列） |
| 有界 / 无界 | 空间范围“多大” | 是否存在一个大区间把集合全部框住 |
| 可测 / 不可测 | 测度能否一致定义 | 是否满足可测条件（如卡拉西奥多里条件） |

### 反例速记（防止误判）

1. (0,1)：有界 + 不可数 + 可测  
2. Z：无界 + 可数 + 可测（测度为 0）  
3. 维塔利集（典型构造）：有界 + 不可数 + 不可测

一句话：
- 可数/不可数看“元素数量类型”
- 有界/无界看“范围是否有限”
- 可测/不可测看“测度结构是否成立”

它们不能互相替代。

---

## 1. 定义：卡拉西奥多里条件

集合 E 称为勒贝格可测集，当且仅当对任意集合 T 都有：
m*(T) = m*(T 交 E) + m*(T 交 E 的补集)

其中 m* 是外测度。

### 本质理解
这个条件表示：E 对任何测试集 T 的切割都“无误差可加”。  
如果切开后外测度正好可加，则 E 是可测的。

对可测集 E，定义其测度为：
m(E) = m*(E)。

## 2. 可测集的核心性质

### 2.1 运算封闭（集合代数性质）
1. 若 E,F 可测，则 E 交 F、E 并 F、E\F、E 的补集都可测
2. 可测集族在可数并、可数交下也封闭

这说明：所有可测集构成一个 sigma-代数。

### 2.2 可数可加性（测度灵魂）
若 {E_n} 两两不交且都可测，则：
m(并 E_n) = sum m(E_n)

这是后续积分与概率的核心结构。

## 3. 测度连续性（集合列极限）

它讲的是**测度 m** 和 **集合极限** 的关系，分成递增列与递减列两种情形。

### 3.1 递增列的连续性（下连续性）
条件：
1. E_1 包含于 E_2 包含于 E_3 ...（递增列）
2. 并集是 E，即 E = 并 E_n

结论：
m(E) = lim m(E_n)（n -> 无穷）

直观理解：
集合越来越大，最终“拼成” E；测度也随之连续逼近到 m(E)。

对应概率论说法：
这就是概率的下连续性。事件列越来越大时，P(E_n) 收敛到极限事件的概率 P(E)。

### 3.2 递减列的连续性（上连续性）
条件：
1. E_1 包含 E_2 包含 E_3 ...（递减列）
2. 交集是 E，即 E = 交 E_n
3. m(E_1) 有限

结论：
m(E) = lim m(E_n)（n -> 无穷）

直观理解：
集合越来越小，最终“缩成” E；只要起点 E_1 的测度有限，测度极限就等于极限集合的测度。

对应概率论说法：
这就是概率的上连续性。

关键注意（高频坑点）：
如果去掉“m(E_1) 有限”，结论可能失败。

反例：
取 E_n = [n, +无穷)。这是递减列，交集是空集，m(空集)=0；  
但每个 E_n 的测度都为无穷，所以 lim m(E_n) 不是 0，连续性失效。

## 4. 易错点清单

1. “可测”不是“形状规则”，而是满足切割可加条件。  
2. 递减连续性不是无条件成立，必须加有限测度前提。  
3. 外测度对任意集合都定义，但可数可加性只在可测集上稳定成立。  
4. 可测集的补集可测，是后续概率事件闭包性质的基础。

## 5. 典型例子

1. 开集、闭集、博雷尔集都是可测集。  
2. 区间类集合（开、闭、半开半闭）都是可测集。  
3. 零测集及其子集都可测。  
4. 康托尔集可测，且测度为 0。

## 6. 与后续章节的衔接

1. 可测函数：要求函数的逆像集合可测。  
2. 勒贝格积分：以可测集与可测函数为定义基础。  
3. 概率论：事件域是 sigma-代数，概率是测度的特例。  
4. 收敛定理（法图、单调收敛、主导收敛）都建立在可测结构上。

## 7. 一页速记

- 可测 = 卡拉西奥多里切割可加  
- 可测集族 = sigma-代数  
- 测度最关键 = 可数可加  
- 递增连续性总成立  
- 递减连续性需有限测度条件  

## 8. 重点证明：有界集合下“内外测度相等”与卡拉条件等价

这一节只证明一个结论（有界情形）：

对有界集合 E，下面两条等价：
1. m_*(E) = m^*(E)
2. 对任意集合 A，有 m^*(A) = m^*(A 交 E) + m^*(A\E)（卡拉条件）

### 8.1 记号与准备

取一个大区间 I，使 E 包含于 I，定义有界情形下内测度：
m_*(E) = m(I) - m^*(I\E)。

其中 m(I) 是通常区间长度，m^* 是外测度。

---

### 8.2 方向一：卡拉条件 => m_*(E) = m^*(E)

这部分最直接。

对卡拉条件取 A=I：
m^*(I) = m^*(I 交 E) + m^*(I\E)。

因为 E 包含于 I，有 I 交 E = E，且 I 可测所以 m^*(I)=m(I)。于是
m(I) = m^*(E) + m^*(I\E)。

移项得
m(I) - m^*(I\E) = m^*(E)。

左边正是 m_*(E)，故
m_*(E) = m^*(E)。

---

### 8.3 方向二：m_*(E) = m^*(E) => 卡拉条件

这部分是核心。

目标：对任意 A 证明
m^*(A) = m^*(A 交 E) + m^*(A\E)。

我们已知次可加性总给：
m^*(A) <= m^*(A 交 E) + m^*(A\E)。

因此只需证明反向不等式：
m^*(A) >= m^*(A 交 E) + m^*(A\E)。

#### 第一步：用“内外相等”得到可测集夹逼

由 m_*(E)=m^*(E) 可得到经典逼近结论：  
对任意 epsilon>0，存在闭集 F 与开集 G，使
F 包含于 E 包含于 G，且 m^*(G\F) < epsilon。

（理解：E 可被“可测的内外壳”任意精细夹住。）

#### 第二步：对可测壳先写等式

因为 F、G 是可测集，对任意 A 有：
m^*(A) = m^*(A 交 F) + m^*(A\F)
m^*(A) = m^*(A 交 G) + m^*(A\G)

并且有包含关系：
A 交 F 包含于 A 交 E 包含于 A 交 G  
A\G 包含于 A\E 包含于 A\F

由单调性可得：
m^*(A 交 F) <= m^*(A 交 E) <= m^*(A 交 G)
m^*(A\G) <= m^*(A\E) <= m^*(A\F)

#### 第三步：误差控制并令 epsilon -> 0

由于 G\F 的外测度小于 epsilon，可控制
|m^*(A 交 G) - m^*(A 交 F)| <= m^*(G\F) < epsilon
|m^*(A\F) - m^*(A\G)| <= m^*(G\F) < epsilon

再结合“壳上的可加等式”，可推出
m^*(A) >= m^*(A 交 E) + m^*(A\E) - 2epsilon。

epsilon 任意，令 epsilon -> 0，得
m^*(A) >= m^*(A 交 E) + m^*(A\E)。

与次可加性方向合并，即得等号：
m^*(A) = m^*(A 交 E) + m^*(A\E)。

卡拉条件成立。

---

### 8.4 结论

在有界情形：
m_*(E)=m^*(E) 与 卡拉可测条件 完全等价。

---

### 8.5 复习骨架（考试写法）

1. 先写已知与目标。  
2. 正向：A=I 代入卡拉条件，一步移项得到内外相等。  
3. 反向：先用内外相等给出 F 包含于 E 包含于 G 且 m^*(G\F)<epsilon。  
4. 对 F、G 写可加等式；用包含关系+单调性夹住 A交E 与 A\E。  
5. 误差估计后令 epsilon->0，得反向不等式。  
6. 与次可加性合并，得到卡拉等式。

## 9. 可测集与卡拉西奥多里条件：一页澄清

这一节只回答一个高频疑问：  
“内外测度相等”和“卡拉条件”到底是什么关系？

### 9.1 两种说法

1. 内外测度说法（直观）  
   m_*(E) = m^*(E)。
   含义：从内部量和从外部量得到同一个值，集合可测。

2. 卡拉西奥多里说法（定义）  
   对任意集合 A，
   m^*(A) = m^*(A 交 E) + m^*(A\E)。
   含义：E 切割任何集合 A 时，外测度完全可加、不产生误差。

### 9.2 什么时候等价

在有界情形（或可先截到有限测度框架）下：
- 内外测度相等  当且仅当  卡拉条件成立。

这就是上一节第 8 节证明的核心结论。

### 9.3 为什么教材统一用卡拉条件

因为卡拉条件是对“任意集合”都可表述的统一定义框架，  
不依赖你先把集合写成有界形式。

所以教材通常直接定义：
- “满足卡拉条件” = “可测集”

而“内外测度相等”更多作为有界情形下的等价判据与直观理解工具。

### 9.4 最短背诵版

1. 直观判据：内外相等可测。  
2. 统一定义：卡拉条件可测。  
3. 有界时两者等价。  
4. 讲全体集合时，用卡拉条件最稳、最统一。

## 10. 排版版速览（30秒）

1. 判定可测：看卡拉切割可加。  
2. 结构性质：sigma-代数 + 可数可加。  
3. 连续性：递增总成立；递减需首项有限测度。  
4. 有界情形：内外相等 与 卡拉条件 等价。  
5. 概率连接：事件域 = sigma-代数，概率 = 测度特例。
