# 微分与不定积分 总目录

本章是实变函数理论的收尾，建立了勒贝格积分框架下的微分与积分关系，推广了经典的微积分基本定理。

## 小节内容概览（已排序）
1. `01 - 维塔利定理.md`：为后续证明提供关键的覆盖工具。
2. `02 - 单调函数的可微性.md`：证明单调函数几乎处处可微，打破黎曼积分的局限。
3. `03 - 有界变差函数.md`：通过若尔当分解，将有界变差函数与单调函数联系起来。
4. `04 - 不定积分.md`：引入绝对连续函数，建立勒贝格框架下的牛顿-莱布尼茨公式。
5. `05 - 斯蒂尔切斯积分.md`：推广黎曼积分，引入关于有界变差函数的积分。
6. `06 - 勒贝格-斯蒂尔切斯测度与积分.md`：将斯蒂尔切斯积分纳入勒贝格测度框架。

---
本章重点：
勒贝格微分定理、绝对连续函数、牛顿-莱布尼茨公式的推广。  
本章难点：
维塔利覆盖引理、康托尔函数的反例理解、L-S测度的构造。
