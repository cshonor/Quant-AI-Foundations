# 可测函数及其性质

## 1. 可测函数的定义（基于可测集的原像）

### 1.1 一般可测空间上的定义
设 (X,A)、(Y,B) 为可测空间，函数 f: X -> Y 称为 A-B 可测，若：
对任意 B 中的可测集 M，都有 f^(-1)(M) 属于 A。

一句话：可测集经过 f 的原像拉回后，仍是可测集。

### 1.1.1 可测集类 与 可测集族（术语对照）
你在第三章看到的“可测集类”和“可测集族”是同一个概念，只是叫法不同。

它们都指：全体勒贝格可测集组成的整体（常记作 L）。
这个整体的本质身份是 sigma-代数，满足：
1. 对补集封闭；
2. 对可数并封闭；
3. 对可数交封闭；
4. 对差集也封闭（由前面性质推出）。

所以后续说“在可测集类中”或“在可测集族中”，数学意义完全等同。

### 1.2 勒贝格可测（最常用）
设 E 是实数轴上的可测集，f: E -> 扩展实数 R∪{+∞,-∞}。
若对任意实数 c，集合
E[f>c] = {x 属于 E: f(x)>c}
都是可测集，则称 f 在 E 上勒贝格可测。

### 1.3 常用等价刻画
对任意 c，下面任一条都可作为可测性判据：
1. E[f>c] 可测；
2. E[f>=c] 可测；
3. E[f<c] 可测；
4. E[f<=c] 可测。

### 1.4 典型例子
1. 连续函数可测；
2. 单调函数可测；
3. 阶梯函数可测；
4. 若集合 A 可测，则其特征函数 chi_A 可测；
5. 狄利克雷函数（有理点取1、无理点取0）可测。

## 2. 简单函数：定义与逼近

### 2.1 简单函数定义
设 E 可测，f: E -> R。若 f 只取有限个值 c1,c2,...,cn，并且每个水平集
Ei = {x 属于 E: f(x)=ci}
都可测，且 E = E1并E2并...并En（两两不交），则 f 称为简单函数。

标准表示：
f(x) = c1*chi_E1(x) + c2*chi_E2(x) + ... + cn*chi_En(x)。

### 2.2 核心性质
1. 简单函数一定可测。
2. 非负可测函数可由非负简单函数列递增逼近：
   存在 phi_n，使 0 <= phi_n <= phi_(n+1)，并且 phi_n(x) 逐点递增到 f(x)。
3. 一般可测函数可拆成 f = f^+ - f^-，其中正部和负部都可由简单函数递增逼近。

这就是勒贝格积分的起点：先对简单函数定义积分，再通过极限扩展到一般可测函数。

## 3. 可测函数的运算封闭性

设 f、g 都是 E 上可测函数，则下列函数（在有定义处）仍可测：
1. 数乘：c*f；
2. 和差：f+g、f-g；
3. 乘积：f*g；
4. 商：f/g（在 g 不为0的点）；
5. 绝对值：|f|；
6. 正部与负部：f^+=max(f,0)，f^-=max(-f,0)。

一句话：可测函数类对四则运算、绝对值和取部操作都封闭。

---

## 4. 可测函数列的极限性质

设 {f_n} 是 E 上可测函数列，则以下函数都可测：
1. sup_n f_n(x)；
2. inf_n f_n(x)；
3. limsup f_n(x) = inf_n sup_{k>=n} f_k(x)；
4. liminf f_n(x) = sup_n inf_{k>=n} f_k(x)。

### 直接推论
1. 若 f_n 逐点收敛到 f，则 f 可测；
2. 若 f_n 几乎处处收敛到 f，则 f 仍可测（改动零测集不影响可测性）。

一句话：可测函数列在上确界、下确界、上极限、下极限、逐点极限、几乎处处极限下都保持可测。

---

## 一页串记
1. 定义：可测集的原像可测；
2. 简单函数：有限取值 + 水平集可测；
3. 逼近：可测函数可由简单函数逼近；
4. 运算：加减乘除、绝对值、正负部都可测；
5. 极限：逐点与几乎处处极限都可测。
