# 可测函数的构造

## 一、卢津定理（Lusin，1912）：可测函数“几乎连续”

### 1) 严格陈述
设 f 是可测集 E（E 包含于 R^n）上的可测函数，且 f 几乎处处有限。
对任意 epsilon > 0，存在闭子集 F_epsilon 包含于 E，使得：
1. m(E \ F_epsilon) < epsilon（挖去的集合测度可任意小）；
2. f 在 F_epsilon 上连续（按子空间拓扑理解）。

一句话：在有限测度框架下，可测函数去掉任意小测度集合后就变成连续函数。

### 2) 证明思路（由简入繁）
1. 简单函数：把每个常值水平集缩小成闭集，有限并下连续性容易控制；
2. 有界可测函数：借助简单函数逼近和叶戈罗夫定理，把逐点收敛强化到坏集外一致收敛；
3. 一般可测函数：写成 f = f^+ - f^-，分别处理正部与负部；
4. 无限测度情形：分解为可数个有限测度块，逐块应用后再拼合。

### 3) 连续延拓推论（全空间逼近）
对任意 epsilon > 0，存在 R^n 上的全局连续函数 g，使
m({x 属于 E: f(x) != g(x)}) < epsilon。

若 E 有界，还可选 g 为紧支集连续函数。

---

## 二、可测函数与连续函数的关系

1. 连续 => 可测：连续函数对开集取原像仍是开集，因此可测。
2. 可测 =/> 连续：如狄利克雷函数、阶梯函数都可测但不连续。
3. 核心桥梁：可测函数是“几乎处处连续”的（卢津定理）。
4. 本质区别：
   - 连续性是逐点局部拓扑性质；
   - 可测性是整体测度性质，允许零测集坏点存在。

---

## 三、连续函数逼近（卢津推论方向）

设 f 在 E 上可测，且几乎处处有限，则：

1. 对任意 epsilon > 0，可找到连续函数 g，使
   ||f-g||_(L1(E)) < epsilon，
   且 m({x 属于 E: f(x) != g(x)}) < epsilon。

2. 若 f 属于 L^p(E)（1 <= p < +无穷），可进一步用紧支集光滑函数逼近（例如 C^infty_c 类函数）。

直观理解：可测函数在测度意义和积分意义下都能被连续函数任意逼近，这正是勒贝格理论可推广黎曼理论的关键之一。

---

## 四、简单函数逼近（构造基础）

### 1) 非负可测函数逼近
若 f >= 0 且可测，则存在非负递增简单函数列 {phi_n}，满足：
0 <= phi_1 <= phi_2 <= ... <= f，
并且对每个 x 属于 E，都有 phi_n(x) -> f(x)。

### 2) 一般可测函数逼近
对任意可测函数 f，存在简单函数列 {phi_n} 使：
|phi_n| <= |f|，
且 phi_n(x) -> f(x)（逐点）。

若 f 有界，可进一步做到一致收敛。

### 3) 意义
1. 简单函数是“有限取值 + 水平集可测”的初等对象；
2. 勒贝格积分先定义在简单函数上，再通过极限扩展到一般可测函数；
3. 这构成了后续单调收敛、主导收敛等定理的技术基础。

---

## 小结（四条核心）
1. 卢津定理：可测函数挖去小测度集合后连续；
2. 关系主线：连续 => 可测；可测函数在几乎处处意义下接近连续；
3. 连续逼近：可测函数可被连续函数在测度/积分意义下逼近；
4. 简单逼近：非负可测函数可由递增简单函数列逼近。
