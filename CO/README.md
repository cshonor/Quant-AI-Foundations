# 凸优化 (Convex Optimization)

凸优化相关笔记。

## 推荐课程：CMU 机器学习中的凸优化

**课程名称**：Convex Optimization / 机器学习中的优化  
**课程编号**：CMU 10-725 / 36-725 / ELE539 / COS512  

面向**机器学习与量化应用**的凸优化实战课，重点是可落地的优化工具与算法，适合快速构建股价预测、量化回测等模型。

### 前置要求
- 编程能力（Matlab / Python / R 等）
- 实分析、微积分、线性代数基础
- 统计/机器学习核心问题、数据结构、计算复杂度、形式化数学思维
- 基础薄弱者可后续补全，建议与讲师沟通

### 核心内容
| 模块 | 内容 |
|------|------|
| 凸优化基础 | 凸集、凸函数、优化问题标准形式 |
| 核心算法 | 梯度下降、次梯度、近端梯度、牛顿法、拟牛顿法、内点法 |
| 对偶理论 | 线性/非线性规划对偶、KKT、对偶方法、ADMM |
| 进阶应用 | 坐标下降、Frank-Wolfe、混合整数规划 |
| 工程导向 | 正则化、投资组合优化等 ML/量化案例 |

### 与国内纯数学教材的对比
| 维度 | CMU 机器学习中的凸优化 | 国内纯数学凸优化教材 |
|------|------------------------|----------------------|
| 定位 | 工程应用，服务于 ML/量化落地 | 数学理论，服务于严谨证明 |
| 节奏 | 先给工具再补原理，见效快 | 先讲理论再推应用，周期长 |
| 侧重 | 梯度下降、对偶、算法实现、代码 | 凸集几何、凸函数证明、对偶空间拓扑 |
| 适合 | 想快速用凸优化做模型的开发者/量化从业者 | 想深入理解优化数学本质的研究者 |

### 建议学习路径
1. **第一步**：跟 CMU 课程学凸函数、梯度下降、对偶基础 → 直接写股价预测代码
2. **第二步**：遇理论卡点时，用国内教材补凸集、凸函数证明细节
3. **第三步**：进阶 ADMM、混合整数规划 → 应用于量化风控与复杂模型

## CMU 10-725 学习计划（按量化+ML目标，优先落地）

### 阶段1：入门与核心工具（优先学习，1–2周）

| 讲座 | 章节 | 核心学习点 | 量化/ML 关联 |
|------|------|------------|--------------|
| 1 | 导论 | 凸优化直觉、课程定位 | 优化=找最优解 ↔ 股价预测=最小化预测误差 |
| 2 | 凸性 I：集合与函数 | 凸集、凸函数定义、几何直观 | 理解线性回归损失为何是碗状，为梯度下降打基础 |
| 3 | 凸性 II：优化基础 | 局部最优=全局最优 | 为何梯度下降在凸问题上能找到最好参数 |
| 4 | 标准问题形式 | 凸优化标准写法、约束处理 | 量化中「风险约束下最大化收益」建模 |
| 5/6 | 梯度下降法 | 原理、步长、收敛性 | **直接写代码**：用梯度下降训练线性回归做股价预测 |
| 7/8 | 次梯度法、近端梯度 | 不可导损失（L1）、加速收敛 | LASSO 选股、防过拟合 |

**目标**：理解凸优化直觉，能用梯度下降训练线性回归模型做股价预测。

### 阶段2：进阶理论与复杂算法（阶段1完成后，2–3周）

| 讲座 | 章节 | 核心学习点 | 量化/ML 关联 |
|------|------|------------|--------------|
| 9/10 | 线性规划对偶 | 弱/强对偶 | 量化中资源分配的底层逻辑 |
| 11 | 一般规划对偶 | 非线性对偶框架 | 投资组合优化、风控模型基础 |
| 12 | KKT 条件 | 最优解充要条件 | 判断收敛、分析策略最优性 |
| 13 | 对偶的应用 | 原问题与对偶对应 | 简化量化里的复杂约束 |
| 14–19 | 牛顿法、拟牛顿、内点法 | 高阶算法、收敛加速 | 加速回测、训练复杂 ML 模型 |
| 20–22 | 对偶方法、ADMM、Frank-Wolfe | 分布式、约束优化 | 多资产组合、DeFi 量化 |

**目标**：掌握约束优化理论，应用于投资组合优化、风控模型。

### 阶段3：拓展与特殊场景（按需学习）

| 讲座 | 章节 | 核心学习点 | 量化/ML 关联 |
|------|------|------------|--------------|
| 23 | 坐标下降法 | 分维度优化、简单高效 | 中低频策略快速回测 |
| 24/25 | 混合整数规划 | 离散决策 | A股仓位选择、策略决策 |

**目标**：处理离散决策，适配 A 股量化策略。

### 学习顺序建议
1. **先冲阶段1**：讲座1–8，重点是梯度下降、凸函数，学完立刻用 Python 写简单股价线性回归模型。
2. **再补阶段2**：处理「最大回撤不超过5%」等约束时，学对偶、KKT、牛顿法。
3. **阶段3按需**：混合整数规划、坐标下降等，在基础模型跑通后按策略需求选择。
