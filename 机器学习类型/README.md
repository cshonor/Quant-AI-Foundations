# 第 2 章预备笔记：机器学习、深度学习与大模型

## 一句话结论

**大模型属于机器学习，是机器学习里规模最大、能力最强的一类。**

## 层级关系（从大到小）

**人工智能 > 机器学习 > 深度学习 > 大模型（Transformer）**

- **人工智能（AI）**：总概念，目标是让机器具备智能行为。  
- **机器学习（ML）**：AI 的方法之一，让机器从数据中学习规律。  
- **深度学习（DL）**：ML 的子集，核心是多层神经网络。  
- **大模型（LLM）**：基于深度学习，通常使用 Transformer，并具有超大数据量与参数规模。  

## 大模型本质在做什么

- 在海量文本上做训练，核心任务常见为：**预测下一个词（token）**。  
- 训练过程是“不断预测 -> 计算误差 -> 参数更新 -> 再预测”。  
- 这种学习方式通常归入**自监督学习**（可视为无监督学习的重要形式）。  

## 和传统机器学习的关系

- 你现在学习的线性回归、随机森林、有监督/无监督学习，是机器学习基础。  
- 大模型是这些思想在“神经网络 + 超大规模数据/算力”条件下的进化结果。  
- 基础越扎实，越容易理解后续的大模型训练、微调和应用。  

## 高频记忆版（背诵）

**机器学习是“从数据学规律”，深度学习是“用神经网络学规律”，大模型是“把深度学习做到超大规模”。**
